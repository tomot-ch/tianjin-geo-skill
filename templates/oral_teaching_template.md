# 口语讲题模板

本模板用于以后根据一道题生成老师风格口语讲解。口语讲解不是随便聊天，而是用自然语言讲清楚“材料—设问—地理含义—因果链—答案”的过程，最后必须落回天津高考规范表达。

## 1. 先翻译题目

这题说的是一个什么事？

```text
这题先别急着看答案，我们先把题目翻译成人话。
它说的是【材料现象/区域背景/图表变化】。
```

## 2. 明确设问

它问的是原因、影响、变化、判断，还是措施？

```text
设问真正要你答的是【题型】。
限定词是【时间/空间/主体/自然或人文/主要原因】。
所以不能答偏到【排除内容】。
```

## 3. 抓材料或图表中最确定的信息

哪些信息最有辨识度？

```text
材料或图表里最确定的信息是：
1. 【最有辨识度的信息】
2. 【能锁定区域/过程/选项的信息】
3. 【能排除干扰项的信息】
```

图表题优先：

- 找占比极高或极低的项。
- 找变化最明显的项。
- 找最能对应典型区域资源或地理过程的项。
- 先确定最稳的，再排剩余项。

## 4. 口语解释

用学生能听懂的话讲清楚。

```text
这句话可以先这么理解：
【口语解释】

它不是在说【易混理解】，而是在提示【真正地理含义】。
```

## 5. 因果链

把中间过程补完整。

```text
【材料条件】
→ 【地理过程1】
→ 【地理过程2】
→ 【结果】
→ 【答案方向】
```

提醒：

- 不能只说结果，要补中间过程。
- 不能从材料条件直接跳到答案。
- 自然地理题尤其要写机制。

## 6. 选项判断

不是看选项本身有没有道理，而是看它能不能解释设问对象。

```text
A 项：是否能解释【设问对象】，原因是……
B 项：可能有相关性，但不是主因，因为……
C 项：与材料限定不匹配，因为……
D 项：最符合材料中的【关键条件】，所以……
```

## 7. 易错提醒

指出学生可能错在哪里。

```text
这题容易错在：
1. 只看选项本身对不对，没有看它是否解释设问对象。
2. 抄材料，没有转成地理含义。
3. 只写结果，没有写中间过程。
4. 忽视区域背景或限定词。
```

## 8. 转成考试语言

给出规范地理表达。

```text
考试里写成：
1. 【短句、分点、贴材料的规范表达】
2. 【短句、分点、贴材料的规范表达】
3. 【短句、分点、贴材料的规范表达】
```

## 9. 可迁移规则

以后遇到同类题怎么想。

```text
同类题以后按这个顺序：
1. 先翻译材料。
2. 明确设问对象。
3. 抓最确定的信息。
4. 补完整因果链。
5. 选项要看能否解释设问对象。
6. 最后转成天津卷规范表达。
```

## 后续迭代接口

新增题目时继续补充：

- 新题型的讲题顺序。
- 新图表题的“最确定项”判断方法。
- 新选择题的选项判断话术。
- 新易错提醒。
- 新口语转考试语言表达。
