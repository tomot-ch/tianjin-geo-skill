# 大题答题风格

本文件用于统一天津高考地理大题的答题风格。核心原则是：不写散句，不堆关键词，要把材料条件转成“规范术语 + 因果过程 + 结果”。

## 1. 基本句式

原因题：
- 条件较好 / 较差，导致某过程增强 / 减弱，从而有利于 / 不利于某结果。
- 受某因素影响，某要素发生变化，进而影响某地理现象。

影响题：
- 有利影响：促进、带动、增加、提高、完善、优化。
- 不利影响：加剧、导致、破坏、减少、制约、威胁。

措施题：
- 针对自然问题：加强治理、恢复植被、合理利用水土资源、完善防灾减灾体系。
- 针对产业问题：优化产业结构、延长产业链、提高科技水平、提升产品附加值。
- 针对区域发展问题：完善基础设施、加强区域协作、因地制宜发展特色产业。

## 2. 大题常用答题链

自然条件链：
- 位置 → 气候 → 水源 → 土壤 → 地形 → 生物 / 生态

农业链：
- 热量、水分、光照、昼夜温差、地形、土壤、水源 → 农作物生长 / 品质 / 熟制 / 产量

工业链：
- 原料、能源、交通、市场、劳动力、科技、政策、产业基础 → 工业布局 / 产业集聚 / 产业升级

城市链：
- 地形、水源、交通、产业、人口、政策 → 城市形成 / 扩展 / 功能提升

交通链：
- 地形、河流、港口、资源、城市、经济水平 → 线路布局 / 运输方式 / 区域联系

## 3. 表达要求

- 一个得分点尽量写成一句完整话。
- 每句话尽量包含条件和结果。
- 不要只写“好”“多”“大”“方便”，要转成规范表达。
- 答案要贴材料，材料没有的信息不要强行扩展。
- 同类要点不要重复，例如“交通便利”和“运输方便”不要当成两个独立点。
- 设问有限定角度时，先按限定筛选材料，再生成答案；不要把自然、人文、经济、生态要点混写。
- 材料有约束条件时，答案必须顺着约束写，例如“资源禁采”后写引进原料、技术提升、延长产业链，不再写继续开采。
- “有利影响/不利影响”“对甲/对乙”“经济/社会/环境”这类设问，要显式分组作答。
- 措施句要有动作对象，优先写“设置、限制、完善、推广、引进、延长、拓展、恢复、加强监管”等可执行动作。

## 4. 天津大题常见语言

- 地形平坦，便于建设和交通线路布局。
- 降水较多，河流补给量大，水资源较丰富。
- 昼夜温差较大，有利于有机质积累，农产品品质较好。
- 交通便利，有利于人流、物流和信息流集散。
- 靠近市场，运输成本较低，有利于产品销售。
- 科技水平较高，有利于提高生产效率和产品附加值。
- 政策支持力度较大，有利于吸引投资和促进产业集聚。

## 5. 2020 年以来真题措辞补充

天津卷综合题答案常见“短句采分”写法：

- 湖泊可在汛期分流洪水，调节地表径流，减轻洪涝压力。
- 地壳抬升使岩层出露地表，为外力侵蚀塑造地貌提供条件。
- 承接产业转移可增加就业机会，改善生产条件，带动劳动力回流。
- 互联网销售可扩大市场范围，降低销售成本，提高经济效益。
- 交通通道建设可加强区域联系，完善交通网络，提高通达性。
- 数据中心集聚有利于企业间信息交流和技术协作，形成规模效益。
- 生态建设用地增加，有利于改善区域生态环境，提高生态系统稳定性。
- 风能、潮流能等清洁能源开发有利于优化能源结构，减少环境污染，保障用能安全。

生成答案时不要机械套整句，应借用其句法：先给条件，再写作用，最后落到明确结果。
