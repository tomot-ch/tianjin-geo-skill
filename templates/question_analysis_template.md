# 题目分析模板

## 1. 题目类型

选择题 / 综合题

## 2. 所属母题

河流 / 气候 / 工业 / 农业 / 城市 / 港口 / 地貌 / 生态

## 3. 材料条件

逐条列出题目给出的显性条件和隐性条件。

示例：
- 显性条件：夏季降水集中。
- 隐性条件：雨水补给增加，河流进入汛期。

## 4. 条件联想

把每个条件转化成地理含义。

示例：
- 降水集中 → 地表径流增加 → 河流补给量增大。
- 地势低平 → 排水不畅 → 易发生洪涝。

## 5. 因果链

写出本题核心推理链。

格式：
- 条件 → 地理过程 → 结果

示例：
- 夏季降水集中 → 地表径流增加 → 河流补给量增加 → 河流流量增大。

## 6. 答案生成

用天津风格语言生成答案。

要求：
- 使用规范术语。
- 每个要点尽量独立。
- 答案来自材料，不空泛扩展。

## 7. 易错点

指出学生容易错在哪里。

常见角度：
- 只抄材料，没有转化为地理含义。
- 只写结论，没有写原因链。
- 混淆流量、水位、含沙量和水能。
- 表达口语化，缺少规范术语。
