# 讲题模板

本模板用于把题目讲给高中生听。语气要自然，逻辑要清楚，不要像论文，也不要像背答案。

## 1. 先说题目在问什么

这道题先别急着写答案，先看它到底问什么。

如果它问“原因”，我们就要找条件，再把条件串成因果链。

如果它问“影响”，我们要先判断影响对象，是经济、社会、生态，还是交通、产业、居民生活。

如果它问“措施”，我们要反过来找问题原因，再对应提出解决办法。

## 2. 再说材料给了什么

材料里的每一句话、每一个图表数据，都可能是出题人给的提示。

可以这样讲：
- 材料说“降水集中在夏季”，这不是让你抄“夏季降水多”，而是提醒你：夏季河流补给增加，径流量变大。
- 材料说“交通干线交会”，不是只写“路多”，而是要转成“交通便利，有利于人流、物流和信息流集散”。

## 3. 再讲如何推出答案

讲题时要把箭头讲清楚。

示例：
- 降水多，不是直接等于洪涝。
- 正确链条是：降水集中且强度大 → 地表径流短时间增加 → 河流水位上涨 → 地势低平地区排水不畅 → 容易发生洪涝。

## 4. 最后总结同类题怎么做

以后遇到同类题，按这四步走：

1. 看设问，确定题目问原因、影响、措施还是比较。
2. 圈材料，把显性条件列出来。
3. 转化条件，把材料语言变成地理术语。
4. 串因果链，用规范语言写成答案。

## 5. 讲题语气要求

- 可以口语化，但不能随便。
- 要让学生知道“为什么这样写能得分”。
- 不要只报标准答案，要讲出材料如何推出答案。
- 遇到易错点，要直接提醒学生下次怎么避开。
