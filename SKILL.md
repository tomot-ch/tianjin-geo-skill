# 天津高考地理 Skill

本文件是仓库的顶层使用说明，用于指导 ChatGPT / Codex 在处理天津高考地理题时如何读取材料、调用链条、生成答案和继续入库。

## 核心目标

本 skill 不是普通知识点索引，而是天津高考地理推理系统。处理题目时必须围绕以下流程：

1. 识别设问限定词。
2. 提取材料有效信息。
3. 判断所属考点和母题。
4. 调用对应解题链。
5. 组织天津卷风格答案。
6. 提炼可迁移规则并沉淀到仓库。

## 读题流程

分析天津地理题时，按以下顺序处理：

1. 看设问动词：原因、影响、措施、意义、比较、判断理由。
2. 看设问限定词：时间、空间、角度、主体、自然或人文范围。
3. 圈材料条件：位置、地形、气候、水源、土壤、资源、交通、市场、劳动力、科技、政策。
4. 转化地理含义：把材料语言改写成规范术语。
5. 串联因果链：条件 → 地理过程 → 结果。
6. 生成答案：短句、分点、贴材料、用天津卷规范表达。
7. 记录迁移：把新母题、新链条、新话术补入对应文件。

## 设问限定词优先级

优先识别：

- 历史上
- 当前
- 只考虑
- 最准确
- 主要
- 从……角度
- 分别说明
- 有利和不利
- 地理意义
- 区位优势
- 技术措施

限定词决定答案边界。题目问“当前”，不要只答历史原因；题目问“自然原因”，不要写市场和政策；题目问“分别说明”，必须分对象作答。

## 选择题判断规则

- “历史形成”看传统功能。
- “当前功能”看现代治理和生态功能。
- “最准确原因”选概括性更强的因素。
- “研发机构”优先技术和人才。
- “人口增长”优先政策、就业、产业、公共服务。
- “产业模式形成”优先产业转型。
- “日出日落”必须结合方位、昼夜长短和经度差异。
- “差值图”要区分变化速度和总量大小。
- 旧工业厂址改造为商业、文化、创意或公园，通常体现城市功能区布局调整。
- 高教科创园、高新区、科创核心区，优先匹配研发、设计、成果转化部门。
- 地层缺失常表示地壳抬升和外力剥蚀。
- 岩浆侵入体晚于其穿过的地层，早于未穿过的上覆地层。
- 河流流速要综合流量、坡降和河道宽窄判断。
- 沿海降水概率要看风是否从海洋吹向陆地以及是否有地形抬升。
- 基岩海岸坡陡水深，利于大型船舶停靠。
- 粮食自给率下降不等于粮食减产，也可能与需求多样化和进口增加有关。
- 生态补偿题必须写上游保护、下游受益、发展受限和协调发展。
- 能源合作题要同时考虑能源安全、经济可行和环境可持续。

选择题必须回到材料和限定词，不要只凭关键词选项。

## 新增读题提醒

分析天津卷时，特别注意以下材料类型：

- 天津地方材料：地面沉降、工业旧址、科创园、滨海新区、传统文化。
- 天津本地生态城、运河文化、京津冀协同发展类题目。
- 图像推理材料：地质图、风场图、日出图、河流剖面图、网络联系图。
- 国家战略材料：京津冀协同、粮食安全、海洋强国、生态补偿、一带一路。
- 城市转型材料：老工业基地、城市更新、产业转型、科创园区。
- 地貌与地质构造判读。
- 人口流动、产业转移与企业回流。
- 公共服务设施密度与城市功能判断。
- 天气形势图、风向、沙尘暴与冷锋判断。
- 地球运动、影子、极昼太阳高度计算。
- 农业区位、互联网销售、特色农业。
- 旅游扶贫、红色旅游、特色畜牧业。
- 北极海冰、冻土、植被变化与温室效应。
- 国际合作、交通建设、内流湖盐度差异。

## 年份权重提醒

2020、2021 年是天津新高考改革后的早期真题，具有主要参考价值，适合补足基础题型链、区域综合题链、地球运动和自然地理链。若 2020、2021 年规则与 2023 年及之后真题规则冲突，应优先采用 2023 年及之后规则。

## 2020、2021 新增读题规则

- 看到“生态城、盐碱荒滩、林地分布在高地”，优先考虑盐碱化与水循环调节。
- 看到“丹霞地貌、红层、陡崖、粤北湿润地区”，优先判断沉积岩、垂直节理、流水侵蚀、地壳抬升。
- 看到“企业迁回、劳动力回流”，要区分主动回流与被动回流，关注市场、政策、产业转移和就业机会。
- 看到“公共服务设施密度”，要根据交通、金融、医疗、文化等设施组合判断城市功能区。
- 看到“高压脊、低压槽、等压线疏密”，要判断天气、风向和锋面活动。
- 看到“雪线、林线、迎风坡、背风坡”，要用水热条件解释垂直带谱范围差异。
- 看到“科技城、科创园、欧洲科技硅谷”，优先政策、人才、研发创新功能。
- 看到“产业体系演变”，要概括由单一专业化向综合性发展。
- 看到“极昼最小太阳高度”，要从极圈纬度与太阳直射纬度关系计算。
- 看到“内流湖一个淡水一个盐湖”，要从入湖河流、出流通道、蒸发和盐分积累解释。

## 新增综合题作答链

城市更新链：
- 旧工业用地衰退 → 改造为商业、文化、创意、休闲空间 → 城市功能优化 → 产业结构升级 → 人居环境改善。

生态补偿链：
- 上游承担生态保护功能 → 发展受限 → 下游受益 → 生态补偿 → 调动保护积极性 → 流域协调发展。

清洁能源链：
- 开发风能、太阳能、潮流能 → 增加清洁能源供给 → 优化能源结构 → 减少污染和碳排放 → 保障能源安全。

## 语言风格补充

优先使用以下表达：

- 体现城市用地功能调整和城市更新。
- 适合布局设计研发等高技术部门。
- 有利于优化产业布局，促进产业分工协作。
- 有利于保护植被，减少水土流失和水体污染。
- 有利于优化能源结构，保障能源供应稳定。
- 有利于调动上游保护生态环境的积极性，促进流域协调发展。
- 距离近，交通便利，有利于承接产业转移。
- 完善交通网络，提高区域通达性，节省出行时间。
- 挖掘历史文化资源，突出地域文化特色。
- 改善旅游服务设施，提高接待能力，扩大客源市场。
- 保护植被，涵养水源，减少水土流失。
- 冰川面积减小，地表吸收太阳辐射增加，温室效应增强。

## 入库分层规则

新增材料进入仓库时必须分层，不把所有内容放进一个文件：

1. 原始试卷、解析、扫描件放入 `papers/年份/` 或 `papers/simulations/`。
2. 逐题提炼放入 `papers/年份/年份_逐题蒸馏.md`。
3. 母题归纳放入 `extracted/`。
4. 条件到答案的推理路径放入 `chains/`。
5. 可直接复用的话术放入 `templates/` 或 `references/tianjin_answer_style.md`。
6. 可复用工作流放入 `prompts/`。

如果材料包含学生答案，只保留答案结构、失分类型和改写示例，不保留姓名、班级、账号、联系方式等隐私信息。

## 2024—2025 高权重规则补充

- 2024、2025 年题目继续强化“材料限定词 + 图表判读 + 区域综合”的组合，不宜只按教材章节套模板。
- 天津地方题常把地方文化、城市更新、河流城市、生态治理与现代功能转型结合起来考。
- 国家战略题常从京津冀协同、雄安新区、生态补偿、能源安全、一带一路和产业转型进入。
- 自然地理图像题要先还原图示含义，再写机制；差值图、风场图、地质图、太阳方位图都不能只凭关键词判断。
- 综合题答案要优先写“材料条件推出的机制”，再写概括性结论。

## 入库原则

1. 一次只处理一个年份或一个专题。
2. 先沉淀母题、链条、话术，再扩展案例。
3. 保留年份、题号、材料关键词和答案依据。
4. 不把解析原文整段堆入知识库，要提炼成可复用规则。
5. 如果新内容与旧文件重复，优先合并为更清晰的规则，不重复造条目。
