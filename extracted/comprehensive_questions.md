# 综合题题型归纳

## 母题一：分主体意义题

触发词：分别说明、对甲地和乙地、对上游和下游、对合作双方。

作答规则：

- 先列主体，再写每个主体的收益或代价。
- 每个主体至少写一个材料条件和一个结果。

常见主体：

- 上游/下游。
- 资源输出地/输入地。
- 投资国/资源国。
- 政府/企业/居民。

## 母题二：区域发展综合题

触发词：京津冀协同、雄安新区、老工业基地转型、乡村振兴、一带一路。

解题链：

区位条件 -> 产业基础或资源条件 -> 政策和交通支撑 -> 产业升级或区域协同 -> 经济、社会、生态意义。

天津话术：

- 有利于优化产业布局，促进区域分工协作。
- 有利于完善基础设施，提高区域通达性。

## 母题三：生态治理综合题

触发词：生态补偿、流域治理、湿地保护、盐碱地修复、生物多样性。

解题链：

生态问题或保护需求 -> 找到责任主体和受益主体 -> 采取工程、生态、管理、政策措施 -> 改善生态环境 -> 促进协调发展。

易错点：

- 只写生态结果，不写治理机制。
- 只写下游受益，不写上游发展受限和补偿必要性。
