# 城市题型归纳

## 母题一：河流与城市形成

来源提示：2024 年天津河流与城市发展。

材料条件：

- 河流交汇、航运便利、商贸集散。
- 城市早期形成、港口城市、因河而生。

解题链：

河流交汇 -> 水运便利 -> 人流物流集散 -> 商贸发展 -> 城市早期形成。

易错点：

- 把历史形成和当前功能混写。
- 只写港口，不写河流交汇和水运集散。

## 母题二：城市功能更新

触发词：老工业基地、旧厂房、文化创意、城市更新、功能区调整。

解题链：

传统工业衰退 -> 土地功能低效 -> 改造为商业、文化、创意、休闲空间 -> 优化城市功能 -> 改善人居环境 -> 推动产业结构升级。

天津话术：

- 体现城市用地功能调整和城市更新，有利于优化城市空间结构。

## 母题三：城市群人口联系

来源提示：2025 年城市群人口联系。

材料条件：

- 中心城市、副中心城市、联系强弱、人口流动方向。
- 经济发展水平、就业机会、交通网络、公共服务。

解题链：

城市等级和经济差异 -> 就业与服务吸引力差异 -> 人口流动规模差异 -> 城市间联系强弱不同。

易错点：

- 只看行政等级，不看经济联系和交通。
- 把人口流动理解为单向流动。
