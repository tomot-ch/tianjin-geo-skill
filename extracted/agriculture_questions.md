# 农业题型归纳

## 母题一：农业区位条件

触发词：农业发展条件、种植优势、特色农业、牧场利用、农产品品质。

材料条件：

- 自然：光照、热量、降水、水源、地形、土壤、昼夜温差。
- 人文：市场、交通、劳动力、技术、政策、品牌、电商平台。

解题链：

自然条件决定能不能种、品质如何；人文条件决定能不能卖、规模如何、收益如何。

天津话术：

- 光照充足、昼夜温差大，有利于有机质积累，提高农产品品质。
- 交通便利、市场广阔，有利于农产品外运和销售。

## 母题二：山地牧场季节利用

来源提示：2025 年天山牧场季节利用。

材料条件：

- 海拔差异。
- 气温、积雪、草场生长期差异。
- 发生时间、消耗时间、转场路线。

解题链：

海拔变化 -> 水热条件变化 -> 草场返青和枯黄时间不同 -> 牧民季节性转场 -> 高海拔多夏季牧场，低海拔多冬季牧场。

易错点：

- 只按海拔机械判断，不结合季节。
- 忽视草场生长期和积雪条件。

## 母题三：农业生态治理

触发词：盐碱地、节水农业、生态农业、退耕还林还草、山西北部生态农业。

解题链：

生态脆弱或资源约束 -> 调整农业结构 -> 推广节水、良种、覆盖、轮作等技术 -> 减少资源消耗和生态破坏 -> 提高农业可持续性。

天津话术：

- 因地制宜发展特色农业，推广节水技术，提高水资源利用效率。
- 调整农业结构，减少不合理开垦，促进生态恢复。
