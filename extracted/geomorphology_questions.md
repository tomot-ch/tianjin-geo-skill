# 地貌与地质题型归纳

## 母题一：岩层新老与地质作用

来源提示：2025 年地质图、植物化石与岩层变质。

材料条件：

- 地质剖面图、岩层时代、化石、岩浆侵入、断层、变质岩。

解题链：

沉积岩层通常下老上新 -> 岩浆侵入晚于被切穿岩层 -> 断层晚于被错断岩层 -> 化石可辅助判断地层时代。

易错点：

- 不看图例时代。
- 混淆裸子植物、被子植物繁盛时代。
- 看到变质岩只写压力，不看岩浆热力作用。

## 母题二：外力地貌形成

触发词：丹霞、冲积平原、三角洲、海岸地貌、风沙地貌。

解题链：

岩性和构造提供基础 -> 地壳抬升或海陆变化改变侵蚀基准 -> 流水、风力、海浪等外力长期作用 -> 形成具体地貌。

天津话术：

- 岩层受流水侵蚀和重力崩塌作用影响，形成陡崖、沟谷等地貌。

## 母题三：地貌与人类活动

触发词：港口岸线、旅游景观、工程建设、生态修复。

作答规则：

- 先写地貌条件。
- 再写对交通、开发、旅游或灾害的影响。
- 最后写合理利用或保护措施。

## 母题四：滑坡、泥石流等地质灾害成因

来源提示：贵州喀斯特山区、山地丘陵区、暴雨多发区。

材料条件：

- 山地、高原、峡谷或坡度较大，地势起伏明显。
- 降水量较多且集中，暴雨多发。
- 地表物质疏松，多碎屑物，或地表岩土破碎。
- 喀斯特地貌、断裂发育、岩体不稳定。

解题链：

坡度大或地势起伏大 -> 岩体稳定性差 -> 降水集中、地表易被侵蚀 -> 地表碎屑物或松散岩土被水流搬运 -> 易发生滑坡、泥石流。

天津话术：

- 山体坡度大，岩体不稳定；降水量较多且集中，地表易被侵蚀；地表物质疏松或多碎屑物，岩体易滑落，易发生滑坡、泥石流等地质灾害。

易错点：

- 只写“坡度大、降水多”，漏写“地表多碎屑物、地表岩土破碎、物质疏松”。
- 把人为原因写进“自然原因”题。
