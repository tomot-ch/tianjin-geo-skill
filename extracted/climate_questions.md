# 气候题型归纳

## 母题一：水热差异与地域分异

来源提示：2024 年澳大利亚水热差异。

材料条件：

- 纬度、海陆位置、大气环流、洋流、地形。
- 只考虑纬度地带性、最准确原因。

解题链：

纬度影响热量 -> 海陆位置和大气环流影响水分 -> 洋流修正沿岸水热 -> 植被和农业类型出现差异。

易错点：

- 忽视“只考虑”限定词。
- 只写洋流，不写更完整的大气环流和海陆位置。

## 母题二：锋面天气

来源提示：2024 年华北冷锋天气。

材料条件：

- 风向转变、等压线、冷空气南下、降温降水大风。

解题链：

冷空气推进 -> 暖空气被迫抬升 -> 冷锋过境 -> 大风、降水、降温 -> 过境后气压升高、气温降低。

## 母题三：气候变化与生态

来源提示：2025 年东山气候变化与植被覆盖。

材料条件：

- 气温、降水趋势线。
- 植被覆盖度、生态系统稳定性。

解题链：

气温和降水变化 -> 水热组合改变 -> 植被生长条件变化 -> 植被覆盖变化 -> 水土保持和生物多样性变化 -> 生态系统稳定性变化。
