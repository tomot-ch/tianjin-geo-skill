# 工业题型归纳

## 母题一：高技术产业区位

来源提示：2024 年广州国际生物岛、2025 年雄安新区高新片区。

材料条件：

- 高校、科研机构、研发平台、人才、政策、基础设施、生态环境。

解题链：

技术密集型产业 -> 需要人才和研发平台 -> 高校科研机构集聚 -> 创新能力增强 -> 促进高技术产业集聚和升级。

易错点：

- 把研发机构当作原料导向型工业。
- 忽略“研发、设计、成果转化”等关键词。

## 母题二：老工业基地转型

来源提示：2025 年齐齐哈尔老工业基地转型。

材料条件：

- 传统工业基础、设备老化、产业结构单一、资源约束、城市更新。

解题链：

传统工业衰退 -> 产业结构单一和环境压力突出 -> 技术改造、延链补链、发展新兴产业 -> 提高附加值 -> 推动城市转型。

天津话术：

- 推动传统产业技术改造，延长产业链，提高产品附加值。

## 母题三：产业转移与回流

触发词：企业迁回、产业承接、劳动力回流、政策扶持。

解题链：

成本变化或市场变化 -> 企业空间布局调整 -> 产业承接地获得就业和产业基础 -> 输出地推动产业升级或功能转换。
