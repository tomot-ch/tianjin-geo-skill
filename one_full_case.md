# 天津高考地理案例

## 题目

分析该地区发展葡萄种植的有利自然条件。

## 第一步：判断题型

属于：

- 农业区位类
- 条件分析类
- 自然条件角度

## 第二步：提取材料信息

材料关键词：

- 光照充足
- 昼夜温差大
- 降水较少
- 地形平坦

## 第三步：匹配知识框架

农业自然区位：

- 气候
- 地形
- 水源
- 土壤

## 第四步：建立因果链

- 光照充足 → 有利于光合作用 → 提高葡萄品质
- 昼夜温差大 → 有利于有机质积累 → 糖分较高
- 地形平坦 → 便于机械化种植

## 第五步：标准答案

1. 光照充足，有利于葡萄进行光合作用，提高产量和品质。
2. 昼夜温差大，有利于有机质积累，提高葡萄糖分含量。
3. 地形平坦，便于机械化生产。
4. 降水较少，病虫害相对较少。

## 学生常见问题

- 只写“气候好”
- 不会展开因果链
- 缺少“有利于”等术语
- 不分点
