## 题目1：各省能源结构占比与省份判读

### 所属专题

- 中国地理
- 能源资源分布
- 图表判读
- 区域资源特征匹配

### 关键词

- 中国能源资源分布
- 黑龙江大庆油田
- 四川盆地天然气
- 甘肃煤炭与黄河上游水能
- 河北煤炭消费
- 图表判读
- 先找最确定项

### 核心知识点

不同省份的能源结构与当地资源禀赋、能源生产和消费结构有关。

判断时不要平均看所有能源类型，应优先找最有辨识度的能源。

### 典型判断线索

黑龙江：

- 石油占比较高。
- 原因：黑龙江有大庆油田，是我国重要石油产区。
- 识别词：石油占比高、大庆油田、东北。

四川：

- 天然气占比较高。
- 原因：四川盆地天然气资源丰富，是我国重要天然气产区。
- 识别词：天然气占比高、四川盆地。

甘肃：

- 煤炭占比较大，同时可能有一定水电占比。
- 原因：甘肃位于西北地区，能源结构中煤炭占比较大；黄河上游地区也有一定水能资源。
- 识别词：煤炭、水电、黄河上游、西北。

河北：

- 煤炭使用占比较大。
- 原因：河北重工业较多，能源消费结构中煤炭比重较高。
- 识别词：煤炭、华北、能源消费。

### 可迁移规则

图表判读题先找最确定项：

1. 某类能源占比极高。
2. 对应典型省份或典型资源。
3. 先锁定最有辨识度的省份。
4. 再排除剩余选项。

### 易错点

- 不要四个柱子平均分析。
- 不要只凭省份印象硬猜。
- 要把能源类型和典型区域资源联系起来。

## 题目2：巴伦支海海洋热量收支与海冰变化

### 所属专题

- 海洋热量收支
- 海冰变化
- 海气热量交换
- 极地海域季节变化
- 大气受热过程辨析

### 关键词

- 海洋热量收支
- 海冰变化
- 海气热量交换
- 大气受热过程辨析
- 海水裸露面积扩大
- 海洋向大气放热

### 核心知识点

海洋热量收支要区分：

1. 海洋吸收太阳辐射。
2. 海洋与大气之间的热量交换。
3. 海冰覆盖对海气热量交换的影响。

题目不能机械套“太阳辐射增强”，要看限定是否指向“海洋与大气之间的热量交换”。

### 核心链条

```text
偏南风增强
→ 暖空气向北输送
→ 北部海冰融化加快
→ 海冰面积减小
→ 海水裸露面积扩大
→ 海气热量交换增强
→ 海洋向大气释放热量增多
```

### 补充链条

海洋从大气吸热：

```text
海面温度低于大气
→ 热量由大气传向海洋
→ 海洋表现为从大气吸热
```

海洋热含量最低月份：

```text
9 月至次年 4 月持续放热
→ 海洋热含量不断下降
→ 到 4 月达到最低
→ 5—8 月开始吸热，热含量回升
```

### 易错辨析

- 把海洋从大气吸热误解成吸收太阳辐射。
- 只看“大气辐射增强”，忽略海冰减少后海气接触面积增大。
- 热含量最低月份不能只看气温最低月份，要看材料中的热量收支过程。

### 口语转书面语

| 口语 | 书面语 |
|---|---|
| 海冰少了，海水露出来更多 | 海冰面积减小，海水裸露面积扩大 |
| 海水和空气更容易交换热量 | 海气热量交换增强 |
| 海水一直往外放热，所以热量越来越少 | 海洋持续向大气释放热量，热含量逐渐降低 |
| 热量从热的地方传到冷的地方 | 海面温度低于大气，热量由大气传向海洋 |

## 题目3：大西洋经向翻转环流与层流下沉区

### 所属专题

- 洋流
- 大西洋经向翻转环流
- 海水性质
- 海水密度、盐度、温度
- 全球变暖影响

### 关键词

- 洋流
- 大西洋经向翻转环流
- 海水密度
- 盐度
- 温度
- 冰川融水
- 全球变暖
- 下沉作用减弱

### 核心知识点

大西洋经向翻转环流的重要机制之一是高纬海域海水下沉。海水是否容易下沉，主要看密度。

### 核心链条

```text
全球变暖
→ 极地冰川融化增多
→ 淡水注入高纬海域增加
→ 表层海水盐度降低
→ 海水密度降低
→ 下沉作用减弱
→ 深层洋流补给减少
→ 大西洋经向翻转环流减弱
```

### 下沉区判断链条

```text
层流下沉区
→ 海水向下沉
→ 下沉通常与海水密度较大有关
→ 密度较大的海水更容易下沉
```

### 易错辨析

- 只看“水位高”，不能直接推出下沉。
- 只看到冰川融化，没有想到盐度和密度。
- 只背洋流名称，不理解盐度、温度、密度、下沉和深层补给之间的机制。

### 口语转书面语

| 口语 | 书面语 |
|---|---|
| 海水往下沉的地方 | 层流下沉区 |
| 密度大就往下走 | 海水密度较大，下沉作用较强 |
| 冰川融化后变成淡水 | 冰川融水注入导致表层海水盐度降低 |
| 上面少了一块补给 | 下沉减弱导致深层洋流补给减少 |

## 题目4：陕西韭园黄土高原水土流失与坝系工程

### 所属专题

- 黄土高原
- 水土流失
- 夏季暴雨
- 坡面径流
- 淤地坝
- 坝系工程
- 水土保持
- 耕地面积增加

### 关键词

- 黄土高原
- 陕西韭园
- 水土流失
- 夏季暴雨
- 坡面径流
- 黄土土质疏松
- 坝系工程
- 淤地坝
- 耕地面积增加

### 区域背景

韭园位于中国陕西省，属于黄土高原相关区域。必须按黄土高原水土流失分析，不能写成尼泊尔。

题干若给出坡度大、地面坡度大于 25 度的地区占比高，说明地形起伏较大，坡面侵蚀条件明显。

### 核心链条1：水土流失

```text
陕西韭园位于黄土高原
→ 黄土土质疏松
→ 坡度大
→ 夏季降水集中，多暴雨
→ 坡面径流强
→ 土壤侵蚀加剧
→ 水土流失严重
```

### 核心链条2：坝系工程增加耕地

```text
在沟谷中修建坝系工程
→ 拦截径流和泥沙
→ 水流速度减慢
→ 泥沙在坝内淤积
→ 沟谷底部逐渐变平
→ 形成坝地或淤地
→ 可利用耕地面积增加
```

### 易错辨析

- 只看坡度，不看降水月份。
- 误把韭园识别为国外地区；本题韭园是中国陕西省地点。
- 认为坝系工程会减少沟谷数量。坝系工程主要改变沟谷内水沙输移过程。
- 不知道“淤地坝”的实际作用。淤地坝不是简单挡水，而是拦水拦沙，促进泥沙淤积，形成可利用坝地。

### 口语转书面语

| 口语 | 书面语 |
|---|---|
| 雨一下大，土就被冲走了 | 降水集中且多暴雨，坡面径流增强，土壤侵蚀加剧 |
| 土本来就松 | 黄土土质疏松，抗侵蚀能力较弱 |
| 坡拿大水一冲，水土就流失了 | 地形坡度大，地表径流流速快，侵蚀作用强 |
| 在沟里修坝，把土拦下来 | 坝系工程拦截径流和泥沙 |
| 淤成一块地，可以种地 | 泥沙淤积形成坝地，使耕地面积增加 |
