# 高中地理核心知识体系（Skill 补强版）

> 本文件用于补充当前天津高考地理 skill 的底层知识框架。它不是替代真题蒸馏，而是帮助模型在遇到新题时有稳定的知识来源。

## 一、自然地理基础

### 1.1 地球的宇宙环境

#### 地球存在生命的条件

外部条件：

- 太阳状态相对稳定，持续提供光热；
- 大小行星绕日公转同向、共面，轨道互不干扰，宇宙环境相对安全。

自身条件：

- 日地距离适中，使地球具有适宜温度；
- 地球质量和体积适中，引力能够吸引并保持大气；
- 有液态水存在；
- 昼夜更替周期适中，有利于温度变化不过分剧烈。

答题链：

```text
日地距离适中 + 大气保温作用 + 昼夜更替适中 → 温度适宜 → 液态水存在 → 有利于生命存在
```

```text
地球质量体积适中 → 引力适中 → 保持大气 → 形成适合生命的大气环境
```

#### 太阳对地球的影响

太阳辐射：

- 维持地表温度；
- 为水循环、大气运动、生物活动和外力作用提供根本动力；
- 为人类生产生活提供光热资源和能源。

太阳活动：

- 黑子活动与气候变化具有相关性；
- 耀斑干扰电离层，影响无线电短波通信；
- 太阳风干扰地球磁场，产生磁暴；
- 高能带电粒子进入两极高空大气，形成极光。

答题提醒：

太阳辐射强调“能量来源”，太阳活动强调“扰动影响”。

### 1.2 地质年代与地理演化

| 地质年代 | 关键特征 | 常见考法 |
|---|---|---|
| 古生代 | 联合古陆形成与分裂，鱼类、两栖动物、蕨类植物繁盛，石炭纪和二叠纪重要成煤 | 生物演化、成煤环境 |
| 中生代 | 燕山运动形成中国地质构造轮廓，裸子植物和恐龙繁盛，侏罗纪重要成煤 | 地质构造、恐龙时代、煤炭形成 |
| 新生代 | 喜马拉雅造山运动，第四纪冰期，哺乳动物和被子植物繁盛，人类出现 | 山脉形成、海平面变化、人类出现 |

答题链：

```text
地壳运动 → 地形抬升 / 构造变化 → 外力作用加强 → 形成特定地貌
```

```text
温暖湿润 + 植物繁盛 + 沉积环境稳定 → 植物遗体堆积 → 成煤
```

### 1.3 森林的生态功能

森林功能：

- 调节气候；
- 涵养水源；
- 保持水土；
- 防风固沙；
- 保护农田；
- 净化空气；
- 吸烟除尘；
- 美化环境；
- 维护生物多样性。

答题链：

```text
森林覆盖率高 → 截留降水、增加下渗 → 涵养水源、调节径流 → 减轻旱涝
```

```text
根系固结土壤 + 枯枝落叶减弱雨滴溅蚀 → 减少水土流失
```

```text
植被提供栖息地 → 物种数量增加 → 维护生物多样性
```

## 二、人文地理核心

### 2.1 人口迁移

影响因素：

- 政治因素；
- 经济因素；
- 社会文化因素；
- 生态环境因素；
- 距离、性别、年龄等其他因素。

答题链：

```text
迁出地就业机会少、收入水平低 → 推力增强 → 人口外迁
```

```text
迁入地经济发达、就业机会多、公共服务完善 → 拉力增强 → 人口迁入
```

```text
生态环境恶化 / 自然灾害频发 → 居住安全下降 → 人口迁出
```

### 2.2 资源环境承载力与人口合理容量

资源环境承载力影响因素：

- 资源；
- 环境水平；
- 经济水平；
- 科技水平；
- 社会文化水平；
- 对外开放程度。

人口合理容量影响因素：

- 资源环境水平；
- 经济水平；
- 科技水平；
- 对外开放程度；
- 消费水平。

答题提醒：

- 资源、科技、经济、开放程度越高，人口容量通常越大；
- 消费水平越高，人均资源消耗越多，人口合理容量可能越小。

### 2.3 城市功能分区

商业区：

- 位于市中心或交通干线附近；
- 土地租金高；
- 交通便捷；
- 昼夜人口数量差别大；
- 经济活动频繁。

住宅区：

- 最广泛的城市土地利用方式；
- 高级住宅区和低级住宅区往往背向发展。

传统工业区：

- 位于城市外围；
- 靠近交通干线；
- 位于盛行风下风向或河流下游。

答题链：

```text
交通便利 + 人流量大 + 地租承受能力强 → 商业区布局
```

```text
污染企业 → 应远离住宅区 → 布局在盛行风下风向、河流下游
```

### 2.4 大都市辐射功能

大都市辐射内容：

- 经济；
- 文化；
- 科技；
- 人才；
- 思想观念。

辐射载体：

- 交通；
- 通信；
- 人员流动。

辐射规律：

- 遵循距离衰减规律；
- 大都市之间辐射范围可能重叠。

答题链：

```text
大都市经济科技实力强 → 通过交通、通信、人员流动向外扩散 → 带动周边地区产业升级和区域发展
```

### 2.5 农业区位

自然因素：

- 气候：热量、日温差、光照、降水；
- 水源；
- 地形；
- 土壤；
- 自然灾害。

社会经济因素：

- 市场；
- 交通；
- 政策；
- 劳动力；
- 技术；
- 土地价格。

答题链：

```text
光照充足 + 昼夜温差大 → 有机质积累多 → 农产品品质好
```

```text
靠近市场 + 交通便利 → 运输成本低 → 产品销售范围扩大
```

### 2.6 工业区位

影响因素：

- 原料；
- 市场；
- 交通；
- 劳动力；
- 能源；
- 环境；
- 科技；
- 政策；
- 产业基础。

答题链：

```text
靠近原料地 → 降低原料运输成本 → 适合原料导向型工业
```

```text
科技人才集聚 → 创新能力强 → 适合研发、设计、高技术产业
```

### 2.7 服务业区位

影响因素：

- 市场；
- 交通；
- 劳动力；
- 政策；
- 地价；
- 地理条件和生态环境。

服务业产品特征：

- 非实物性；
- 不可储存性；
- 生产与消费同时发生。

答题链：

```text
人口密集 + 消费能力强 + 交通便利 → 客流量大 → 适合布局服务业
```

### 2.8 交通运输与区域发展

交通对区域发展的影响：

- 提高区域通达性；
- 促进资源开发和产业布局优化；
- 扩大市场范围；
- 加快城镇化进程；
- 巩固国防；
- 促进区域间社会文化交流。

区域发展对交通的影响：

- 提供资金、技术保障；
- 形成运输需求，是交通发展的根本动力。

答题链：

```text
交通改善 → 通达性提高 → 要素流动增强 → 市场范围扩大 → 区域竞争力提升
```

## 三、区域可持续发展

### 3.1 黄土高原水土流失

自然原因：

- 坡度大，地表径流流速快；
- 夏季降水集中且多暴雨；
- 黄土质地疏松，垂直节理发育；
- 植被覆盖率低，保持水土能力弱。

人为原因：

- 过度农垦；
- 过度放牧；
- 过度樵采；
- 不合理工程建设，如开矿、修路。

答题链：

```text
坡度大 → 地表径流速度快 → 流水侵蚀强 → 水土流失严重
```

```text
夏季降水集中且多暴雨 → 短时间径流量大 → 冲刷力强 → 水土流失严重
```

```text
黄土质地疏松、垂直节理发育 → 抗侵蚀能力弱 → 易被流水侵蚀
```

```text
过度农垦、放牧、樵采 → 植被破坏 → 地表裸露 → 水土流失加剧
```

治理措施：

- 退耕还林还草；
- 植树种草，恢复植被；
- 修筑梯田；
- 打坝淤地；
- 等高耕作；
- 少耕免耕；
- 加强生态保护和监管。

## 四、国家安全与生态环境

### 4.1 海洋空间资源与国家安全

海洋空间资源类型：

- 海岸带与海岛空间资源；
- 海面空间资源；
- 海中与海底空间资源。

开发优点：

- 节省土地资源；
- 无需迁移人口；
- 环境相对安全稳定。

利用难度：

- 黑暗；
- 高压；
- 低温；
- 缺氧；
- 腐蚀；
- 生物附着。

国家安全意义：

- 保障海岸带与岛屿安全；
- 维护海洋资源开发安全；
- 保护海洋通道安全；
- 提升遏制管控海上军事危机的能力。

### 4.2 碳排放与碳减排

核心概念：

- 碳循环；
- 碳平衡；
- 温室效应；
- 碳汇；
- 碳源。

碳排放影响：

- 海平面上升，淹没沿海低地和岛屿；
- 极端天气事件频发；
- 改变农业生产条件，影响粮食安全；
- 加剧疾病传播，威胁人类健康。

碳减排措施：

- 更新发展观念；
- 政策指引，如碳税、碳交易；
- 技术支持，如清洁能源、碳捕获；
- 加强国际合作。

答题链：

```text
化石能源大量燃烧 → 二氧化碳排放增加 → 温室效应增强 → 全球变暖 → 海平面上升和极端天气增加
```

```text
发展清洁能源 + 提高能源利用效率 + 增加碳汇 → 减少碳排放 → 缓解全球变暖
```

### 4.3 污染物跨境转移

转移途径：

- 大气循环；
- 水循环；
- 国际贸易；
- 非法运输。

应对措施：

输出国：

- 控制污染物排放；
- 严格执行国际公约；
- 加强污染途径管控。

输入国：

- 加强跨境污染监测；
- 完善法律法规；
- 建立预警机制。

国际层面：

- 健全合作机制；
- 共享信息；
- 协商、仲裁解决争端；
- 发挥国际组织作用。

### 4.4 自然保护区与生态安全

我国自然保护地体系：

| 类型 | 主要保护对象 | 分区管控 |
|---|---|---|
| 国家公园 | 国家代表性自然生态系统 | 核心保护区 + 一般控制区 |
| 自然保护区 | 典型生态系统、珍稀濒危物种、特殊自然遗迹 | 核心保护区 + 一般控制区 |
| 自然公园 | 重要生态系统、自然遗迹和景观 | 一般控制区 |

设立意义：

- 保护自然本底；
- 贮备物种资源；
- 维护生物多样性；
- 涵养水源、净化空气；
- 提供科学研究场所。

### 4.5 环境保护与国家安全

环境问题产生原因：

- 人类索取资源的速度超过环境供给能力；
- 人类排放废弃物的数量超过环境自净能力。

环境保护措施：

- 改善大气环境质量；
- 保护水源；
- 修复土壤污染；
- 推动绿色发展方式和生活方式；
- 加快生态保护与修复。
