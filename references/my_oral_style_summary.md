# 老师个人口语讲题风格总结

本文件用于沉淀老师真实聊天记录和语音转文字中的讲题习惯。重点不是机械模仿口头禅，而是提炼可迁移的讲题动作。后续每新增一批聊天记录，都继续追加，不覆盖原有规则。

## 1. 总体讲题风格

老师讲题时不是直接报答案，而是先带学生把题目“翻译成人话”。

常见流程：

1. 先说明题目说了一个什么事。
2. 再明确题目到底问什么。
3. 再找材料或图表中最关键的信息。
4. 再判断选项能不能解释题目问的现象。
5. 最后把口语解释转换成高考规范表达。

核心原则：

- 不要一上来背知识点。
- 不要一上来逐个硬排选项。
- 不要只看选项本身对不对。
- 要看选项能不能解释题干中的现象。
- 讲给学生听时可以口语化。
- 最后必须落回规范地理语言。
- 真实聊天记录入库前必须匿名化，只保留教学价值。

## 2. 选择题讲解核心规则：先翻译材料，再判断选项

讲选择题时，不要先背知识点，也不要先看选项。

第一步先带学生回到材料，说明题目到底描述了什么现象、什么变化、什么因果过程。

常用讲题动作：

- 先把题干翻译成人话。
- 再明确设问对象。
- 再问“哪个选项最能解释这个对象”。
- 最后判断选项是否是主要原因、直接原因或限定条件。

示例占位：

- “这题你先别急着看选项，先看材料说了个什么事。”
- “这个题不是让你背概念，而是让你根据材料推出原因。”
- “判断选择题，不是看哪个选项听起来对，而是看哪个选项最能解释材料里的现象。”
- “这个选项本身可能有道理，但它不是这道题问的主要原因。”

适用题目：

- 题目2：巴伦支海海洋热量收支。
- 题目3：大西洋经向翻转环流。
- 题目4：陕西韭园水土流失。

## 3. 图表题讲解规则：先找最确定项，再排除剩余项

遇到图表判读题时，不要平均用力。

先找最有辨识度、最确定的信息，再用排除法确定剩下的。

讲题动作：

- 找占比最高、差异最大、最有区域辨识度的信息。
- 用典型资源或典型区域先锁定一个对象。
- 再处理剩余选项，避免四个柱子同时硬对。

示例占位：

- “这四个里面，你先找最好确定的。”
- “不要四个柱子一起硬对，先看哪个特征最明显。”
- “最确定的先定下来，剩下的再排。”
- “这类题不是每个信息都要一样重要，先抓最有辨识度的那个。”

题目1中的具体表现：

- 石油占比特别高，优先联想黑龙江和大庆油田。
- 天然气占比高，优先联想四川盆地。
- 水电占比明显，可联想西南地区或黄河上游等水能资源较丰富地区。
- 煤炭占比高，可以联想华北、西北、黄土高原等能源消费或煤炭资源特征。

## 4. 纠错规则：先承认相关性，再指出不是主因

学生问“这个选项能不能选”时，不要直接说“不能”。

如果学生思路有一定相关性，先承认它“可能有点关系”，再指出它不是本题的主要限定条件或最直接链条。

讲题动作：

- 先判断学生答案是否与材料有关。
- 再区分“有关”和“主因”。
- 再回到设问对象和材料限定。
- 最后给出更直接的因果链。

示例占位：

- “你这个方向不是完全没关系。”
- “这个选项本身可能也能解释一部分现象。”
- “但是这道题问的是主要原因。”
- “你要看题干到底让你解释哪个现象。”
- “它和题目有关，但不是最直接的那条链。”

适用题目：

- 题目2中，学生问其他选项能不能选时，可先承认其相关性，再强调本题主要看海冰减少后海气接触面积扩大。

## 5. 专业概念翻译规则：先讲成人话，再讲术语

遇到专业概念，不要直接背定义。先把概念翻译成人话，让学生知道它在题目里具体表示什么过程，然后再回到术语。

### 层流下沉区

口语解释：
就是海水往下沉的地方。

规范理解：
海水密度增大后更容易下沉，形成深层洋流的重要补给区域。

### 坝系工程 / 淤地坝

口语解释：
就是在沟里修坝，把水和泥沙先拦下来。水慢下来以后，泥沙沉积，慢慢形成一块比较平的地。

规范理解：
坝系工程拦截径流和泥沙，降低流速，促进泥沙淤积，形成坝地，增加可利用耕地面积。

### 海洋向大气放热

口语解释：
海水把热量传给空气。

规范理解：
海洋与大气之间发生热量交换，海洋向大气释放热量。

## 6. 因果链讲解规则：不能只说结果，要补中间过程

讲自然地理题时，不能只报结果。要把中间过程补出来，让学生知道为什么。

讲题动作：

- 指出学生是否跳过中间过程。
- 把“条件 → 过程 → 结果”拆开讲。
- 最后压缩成天津卷规范表达。

示例占位：

- “中间少了一步。”
- “你这个说法方向对，但是链条不完整。”
- “不是直接从 A 到 C，中间还有一个 B。”
- “这题要把过程写出来，不然答案太空。”
- “高考答案不是只写现象，要写机制。”

示例链条：

```text
海冰面积减小
→ 海水裸露面积扩大
→ 海洋与大气直接接触面积增大
→ 海气热量交换增强
→ 海洋向大气释放热量增多
```

## 7. 口语转书面语规则

讲题时可以口语化，但最终必须转成高考能得分的表达。

| 口语说法 | 规范表达 |
|---|---|
| 水流没劲了 | 流速减慢，搬运能力下降 |
| 泥沙堆下来了 | 泥沙沉积 |
| 海水露出来更多了 | 海水裸露面积扩大 |
| 海水和空气更容易交换热量 | 海气热量交换增强 |
| 土被水冲走了 | 坡面径流增强，土壤侵蚀加剧 |
| 把土拦下来，淤成一块地 | 拦截径流和泥沙，促进泥沙淤积，形成坝地 |
| 雨一下大，冲得就厉害 | 降水集中且多暴雨，地表径流增强，侵蚀作用加强 |
| 密度大就往下沉 | 海水密度增大，下沉作用增强 |
| 淡水一多，盐度就低了 | 淡水注入增加，表层海水盐度降低 |

## 8. 天津卷提醒方式

个人口语讲题必须保留天津卷采分意识：

- 贴材料：答案不能脱离题干条件。
- 短句分点：避免长句堆叠。
- 术语规范：把口语解释落回地理术语。
- 因果完整：条件、过程、结果尽量不断链。
- 限定词优先：题目限定自然、人文、当前、历史、分别说明时，答案边界必须跟上。
