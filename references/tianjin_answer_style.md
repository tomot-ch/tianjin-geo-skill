# 天津卷答案风格

本文件用于约束综合题答案表达，重点解决“会想但写不得分”的问题。

## 总原则

- 短句分点，避免长段堆砌。
- 贴材料写，不空泛背模板。
- 每个要点至少包含一个明确条件或机制。
- 优先使用规范地理术语。
- 不写绝对化、情绪化和口语化表达。

## 推荐句式

- ……条件较好，有利于……
- ……增加，使……增强，从而……
- ……减少，导致……下降，进而……
- ……完善，提高……效率，促进……
- ……受限，需要通过……实现……
- ……既能……，又能……

## 高频可迁移表达

- 体现城市用地功能调整和城市更新。
- 完善交通网络，提高区域通达性。
- 促进产业分工协作，优化产业布局。
- 提高资源利用效率，减少污染物排放。
- 增强生态系统稳定性，保护生物多样性。
- 调动上游生态保护积极性，促进流域协调发展。
- 优化能源结构，保障能源供应稳定。
- 挖掘历史文化资源，突出地域文化特色。
- 延长产业链，提高产品附加值。

## 失分表达替换

| 口语或弱表达 | 建议替换 |
|---|---|
| 水很多 | 水资源丰富；河流补给量较大 |
| 人多 | 劳动力资源丰富；市场需求量大 |
| 路好 | 交通便利；交通网络较完善 |
| 环境变差 | 生态环境退化；生物多样性下降 |
| 赚钱 | 增加居民收入；促进区域经济发展 |
| 风大 | 风力较强；风能资源较丰富 |
| 地方有名 | 地域文化特色鲜明；旅游吸引力较强 |

## 禁忌

- 不把“影响”题写成单纯措施。
- 不把“措施”题写成单纯意义。
- 不把“自然原因”写入市场、政策、交通。
- 不把“当前功能”写成历史功能。
- 不把“最准确原因”写成单一关键词。
