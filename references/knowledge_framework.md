# 天津高考地理知识框架

本文件用于把天津卷常见材料转化为可复用的做题框架。使用时先看题型入口，再看材料条件，最后调用对应链条和答案模板。

## 总体能力结构

天津卷地理题通常考四层能力：

1. 材料读取：从文字、图表、地图、剖面图、统计图中提取有效条件。
2. 题型识别：判断设问是原因、影响、措施、意义、区位、比较、判断理由还是图表判读。
3. 地理推理：把材料条件转化为地理过程，形成“条件 -> 过程 -> 结果”。
4. 答案表达：用短句、分点、贴材料的天津卷语言组织答案。

## 高权重年份使用规则

- 2023 年及之后真题权重最高。
- 2020 年及之后真题具有主要参考价值。
- 2019 年可作为改革前后的衔接参考。
- 2019 年之前真题只作补充，不作为主要规则来源。
- 如果旧题风格与 2023 年及之后真题冲突，优先采用新题规则。

## 题型入口

### 选择题

先判断限定词：

- “历史形成”：看传统区位、早期交通、河流航运、商贸集散。
- “当前功能”：看现代治理、生态、防洪、景观、产业转型。
- “只考虑”：只在题干限定范围内作答，不外扩。
- “最准确”：优先选解释范围更完整、概括层级更高的选项。
- “主要因素”：找材料中最直接、最能控制结果的条件。

再判断材料类型：

- 差值图：先判断正负值含义，再区分变化速度和总量大小。
- 风场图：看风向转变、风速差异、锋面两侧气团性质。
- 地质图：看岩层新老、侵入切穿关系、化石时代和变质线索。
- 太阳方位图：看昼夜长短、太阳直射点、影子方向和拍摄方向。
- 网络联系图：看线条粗细、方向、节点等级和距离阻力。

### 综合题

先拆设问：

- 原因题：自然条件、社会经济条件、政策条件、技术条件、历史基础。
- 影响题：有利、不利；生态、经济、社会；短期、长期。
- 措施题：工程、技术、管理、政策、生态、产业。
- 意义题：生态意义、经济意义、社会意义、区域协同意义、国家安全意义。
- 区位题：自然区位、交通区位、市场区位、产业基础、科技人才、政策。
- 分主体题：按国家、企业、当地居民、上游、下游、输出地、输入地分别写。

## 专题框架

### 自然地理

- 气候：纬度、海陆位置、大气环流、地形、洋流、人类活动。
- 河流：补给、流量、水位、含沙量、结冰期、水能、洪涝、河流地貌。
- 地貌：内力作用、外力作用、岩性、构造、气候和时间尺度。
- 生态：气候变化、植被覆盖、水土保持、生物多样性、生态系统稳定性。
- 地球运动：昼夜长短、太阳方位、正午太阳高度、经度时差。

### 人文地理

- 农业：水热土光、地形、水源、市场、交通、技术、政策、品牌。
- 工业：原料、能源、市场、交通、劳动力、科技、产业基础、政策。
- 城市：城市形成、功能区布局、城市更新、人口流动、公共服务。
- 交通港口：水深岸线、腹地、交通网络、货源、城市依托、政策。
- 区域发展：产业转型、生态治理、区域协同、国家战略、对外合作。

## 答案生成规则

1. 每个要点独立成句，方便阅卷按点给分。
2. 优先写材料里直接给出的条件。
3. 把口语表达转成地理术语。
4. 因果链不要断：不能只写“有利于发展”，要写为什么有利。
5. 不写与设问限定无关的内容。
