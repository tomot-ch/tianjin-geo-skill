# 完整案例：河流流量变化题

## 题目材料

某河流位于季风气候区，流域内夏季降水较多，降水集中。该河流流域面积较大，支流较多，干流下游流经平原地区。

## 设问

分析该河流夏季流量大的原因。

## 母题识别

河流补给与流量变化。

## 材料条件

- 夏季降水多
- 流域面积较大
- 支流较多

## 条件联想

- 夏季降水多 → 雨水补给多
- 支流较多 → 汇水量大
- 流域面积较大 → 集水范围广

## 因果链

夏季降水集中 → 地表径流增加 → 河流补给量增加 → 河流流量增大。

支流较多、流域面积较大 → 汇水范围广、汇水量大 → 干流流量较大。

## 天津风格答案

夏季受降水影响明显，降水量较大，地表径流增加，河流补给量增多；流域面积较大，支流较多，汇水量大，因此夏季河流流量较大。

## 讲题稿

这道题本质上不是问你背哪个知识点，而是让你看材料里的条件。

材料里说夏季降水多，那第一反应就是雨水补给增加；又说支流多、流域面积大，那就是汇水范围广、汇水量大。所以答案要写成一条链：降水多，径流增加，补给增多，流量变大。

注意不要只写“因为夏季下雨多”。这句话太短，得分点不够。天津卷综合题更喜欢看到完整表达：夏季降水较多，地表径流增加，河流补给量增大，河流流量较大。

以后遇到河流流量题，先问自己三件事：

1. 水从哪里来，是降水、冰雪融水，还是地下水？
2. 汇水条件怎么样，流域面积大不大，支流多不多？
3. 材料有没有提示季节变化，比如夏季、春季、汛期、枯水期？

把这三件事想清楚，河流流量题基本就能写出稳定答案。
