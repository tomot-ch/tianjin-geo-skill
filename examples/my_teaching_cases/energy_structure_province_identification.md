# 题目编号

题目1

# 题目主题

各省能源结构占比与省份判读

# 原始讲解来源

聊天记录 / 语音转文字 / 截图整理

# 隐私处理

已删除学生姓名、头像、学校、手机号、微信号等信息。

# 讲题风格线

本题体现出的讲题动作：

- 图表题先找最确定项，不要平均用力分析所有柱子。
- 先抓最有辨识度的能源类型，再匹配典型省份。
- 用已确定省份排除剩余选项。
- 把“看图猜省份”转成“能源结构特征匹配区域资源禀赋”。

# 知识点线

所属专题：

- 中国地理。
- 能源资源分布。
- 图表判读。
- 区域资源特征匹配。

核心知识点：

- 黑龙江石油资源突出，可联系大庆油田。
- 四川盆地天然气资源丰富。
- 甘肃煤炭占比较大，黄河上游地区有一定水能资源。
- 河北重工业较多，能源消费中煤炭比重较高。

核心因果链：

```text
能源结构图表
→ 找占比最突出、最有辨识度的能源
→ 联系典型资源产区或消费结构
→ 先确定最稳省份
→ 排除剩余选项
```

# 口语转书面语

| 口语说法 | 规范表达 |
|---|---|
| 先看最明显的那个柱子 | 优先识别图表中最具辨识度的能源占比特征 |
| 石油高就先想大庆 | 石油占比较高，可联系黑龙江大庆油田 |
| 天然气高就想四川盆地 | 四川盆地天然气资源丰富，天然气占比较高 |
| 不要四个一起硬看 | 图表判读应先确定典型项，再排除剩余项 |

# 易错点

- 四个柱子平均分析，找不到突破口。
- 只凭省份印象硬猜，没有联系能源资源分布。
- 忽略能源生产结构和能源消费结构的差异。

# 可迁移规则

- 图表题先找最确定项。
- 区域判读题要把图表特征和典型区域资源联系起来。
- 能源结构题优先关注石油、天然气、水电、煤炭等高辨识度指标。
