# 基础知识调用示例

## 示例一：水土流失成因

### 题目设问

分析某区域水土流失严重的自然原因。

### 材料条件

- 坡度较大；
- 夏季降水集中；
- 多暴雨；
- 土质疏松；
- 植被覆盖率低。

### 知识调用

调用：

- `references/core_高中地理核心知识体系.md`
- `chains/core_自然地理基础链.md`

### 答题链

```text
坡度大 → 地表径流速度快 → 冲刷能力强 → 水土流失严重
```

```text
降水集中且多暴雨 → 短时间径流量大 → 流水侵蚀强 → 水土流失严重
```

```text
土质疏松 → 抗侵蚀能力弱 → 易被流水侵蚀
```

```text
植被覆盖率低 → 保持水土能力弱 → 水土流失加剧
```

### 标准答案

① 地形坡度较大，地表径流速度快，流水冲刷能力强。  
② 夏季降水集中且多暴雨，短时间内径流量大，侵蚀作用强。  
③ 土质疏松，抗侵蚀能力弱，易被流水侵蚀。  
④ 植被覆盖率较低，涵养水源、保持水土能力弱。

### 学生版讲解

这题可以记成四个词：坡大、雨急、土松、草少。  
坡大所以水流得快，雨急所以冲得猛，土松所以扛不住，草少所以没人帮它“抓土”。  
标准答案就把这四点展开写。

## 示例二：高技术产业区位

### 题目设问

说明某科创园适合发展高技术产业的区位条件。

### 材料条件

- 高校和科研院所集中；
- 交通便利；
- 政策支持；
- 靠近大城市；
- 信息通达。

### 答题链

```text
高校科研院所集中 → 科技人才丰富 → 研发创新能力强 → 适合高技术产业
```

```text
交通便利 + 信息通达 → 要素流动效率高 → 有利于企业协作和成果转化
```

```text
政策支持 → 资金、土地、基础设施保障 → 产业发展环境较好
```

### 标准答案

① 高校和科研院所集中，科技人才丰富，研发创新能力强。  
② 交通便利、信息通达，有利于人才、资金、技术等要素流动。  
③ 政策支持力度大，有利于完善基础设施和优化产业发展环境。  
④ 靠近大城市，市场广阔，生产性服务业和产业协作条件较好。

### 学生版讲解

高技术产业不是看哪里便宜，而是看哪里“脑子多、信息快、政策好、协作强”。  
所以这类题优先写人才、科技、交通信息、政策和产业基础。
