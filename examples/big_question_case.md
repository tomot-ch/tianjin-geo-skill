# 综合题案例：生态补偿意义

## 题目材料

某流域上游承担水源涵养和生物多样性保护功能，产业开发受到限制；下游城市用水和生产活动受益于上游生态保护。地方建立跨区域生态补偿机制。

## 设问

说明建立生态补偿机制的意义。

## 母题识别

生态补偿分主体意义题。

## 材料条件

- 上游承担生态保护功能。
- 上游产业开发受限。
- 下游获得清洁水源和生态服务。
- 建立补偿机制。

## 解题链

上游保护生态 -> 发展受限、保护成本增加 -> 下游受益 -> 通过资金或产业补偿调动上游保护积极性 -> 促进流域协调发展。

## 天津风格答案

有利于补偿上游生态保护成本，调动上游保护生态环境的积极性；有利于保障下游用水安全和生态安全；有利于协调上下游利益关系，促进流域可持续发展。
