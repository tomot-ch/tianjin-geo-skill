# 人文地理基础链

## 1. 人口迁移链

```text
迁出地就业机会少、收入水平低 → 生活改善空间有限 → 人口外迁
```

```text
迁入地经济发展水平高、就业机会多 → 收入预期提高 → 人口迁入
```

```text
迁入地教育、医疗、公共服务完善 → 生活质量较高 → 人口吸引力增强
```

```text
生态环境恶化 / 自然灾害频发 → 居住安全下降 → 人口迁出
```

## 2. 城市功能分区链

```text
城市中心交通便利、人流量大 → 商业收益高 → 商业区集聚
```

```text
交通干线交会处 → 可达性强 → 商业和服务业活动集中
```

```text
工业污染较大 → 远离居住区 → 布局在城市外围、盛行风下风向或河流下游
```

```text
城市旧工业区衰退 → 土地功能调整 → 商业、文化、创意、休闲空间发展 → 城市更新
```

## 3. 农业区位链

```text
光照充足 + 昼夜温差大 → 有机质积累多 → 农产品品质好
```

```text
热量充足 → 生长期长 / 熟制提高 → 农业生产潜力大
```

```text
水源充足 → 灌溉条件好 → 农业生产稳定
```

```text
靠近市场 → 运输距离短 → 产品新鲜度高、成本低 → 商品农业发展
```

```text
交通便利 → 农产品外运方便 → 市场范围扩大
```

## 4. 工业区位链

```text
靠近原料地 → 原料运输成本低 → 原料导向型工业布局
```

```text
靠近市场 → 产品运输成本低 → 市场导向型工业布局
```

```text
能源丰富 → 动力供应充足 → 高耗能工业发展
```

```text
科技人才集聚 → 研发能力强 → 高技术产业和研发设计部门集聚
```

```text
产业基础好 → 协作条件强 → 产业集群形成
```

## 5. 服务业区位链

```text
人口密集 + 消费水平高 → 市场需求大 → 服务业集聚
```

```text
交通便利 → 客流量大 → 商业服务业发展
```

```text
信息通达 + 专业人才集中 → 生产性服务业发展
```

## 6. 交通与区域发展链

```text
交通建设 → 通达性提高 → 人流、物流、资金流、信息流增强 → 区域联系加强
```

```text
交通改善 → 运输成本降低 → 市场范围扩大 → 产业发展条件改善
```

```text
区域经济发展 → 运输需求增加 → 资金和技术保障增强 → 推动交通建设
```
