# 城市因果链库

## 城市形成

触发条件：
- 河流交汇、港口、交通节点、地形平坦、商贸集散。

因果链：
- 河流交汇 -> 水运便利 -> 人流物流集散 -> 商贸发展 -> 城市形成和发展。
- 地形平坦 -> 建设成本较低 -> 交通线路布局便利 -> 城市扩展条件较好。

天津风格表达：
- 天津位于海河水系下游，多条河流交汇，水运条件较好，有利于人员和物资集散，促进城市早期形成和商贸发展。

## 城市更新

触发条件：
- 老工业区、旧厂房、低效用地、文化创意、商业休闲。

因果链：
- 传统工业衰退 -> 土地利用效率下降 -> 更新改造 -> 发展商业、文化、创意和休闲功能 -> 优化城市空间结构。

天津风格表达：
- 体现城市用地功能调整和城市更新，有利于优化城市功能布局，改善人居环境。

## 人口流动与城市联系

触发条件：
- 城市群、中心城市、副中心城市、人口联系、交通网络。

因果链：
- 经济发展水平高 -> 就业机会多 -> 公共服务完善 -> 人口吸引力增强。
- 交通网络完善 -> 出行成本降低 -> 城市间联系增强 -> 人口流动规模增大。
