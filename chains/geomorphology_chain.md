# 地貌与地质因果链库

## 岩层新老关系

触发条件：
- 地质剖面图、岩层、断层、岩浆侵入、化石。

因果链：
- 沉积岩层连续沉积 -> 通常下老上新。
- 岩浆侵入体切穿某岩层 -> 侵入体晚于被切穿岩层。
- 断层错断岩层 -> 断层晚于被错断岩层。
- 化石类型 -> 指示地质时代 -> 辅助判断岩层新老。

## 变质作用

触发条件：
- 片麻岩、变质岩、岩浆侵入、构造挤压。

因果链：
- 岩浆侵入 -> 周围岩石受高温影响 -> 矿物成分和结构改变 -> 形成变质岩。
- 构造挤压 -> 岩石受高压作用 -> 发生变质。

天津风格表达：
- 岩层受岩浆活动产生的高温影响，原有岩石矿物成分和结构发生改变，形成变质岩。

## 外力地貌

触发条件：
- 流水侵蚀、流水堆积、风力作用、海浪作用。

因果链：
- 地势落差大 -> 流水下切侵蚀强 -> 形成峡谷。
- 河流流速减慢 -> 泥沙沉积 -> 形成冲积平原或三角洲。
- 干旱多风 -> 风力搬运和沉积 -> 形成沙丘等风积地貌。
