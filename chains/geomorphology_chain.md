# 地貌与地质因果链库

## 岩层新老关系

触发条件：
- 地质剖面图、岩层、断层、岩浆侵入、化石。

因果链：
- 沉积岩层连续沉积 -> 通常下老上新。
- 岩浆侵入体切穿某岩层 -> 侵入体晚于被切穿岩层。
- 断层错断岩层 -> 断层晚于被错断岩层。
- 化石类型 -> 指示地质时代 -> 辅助判断岩层新老。

## 变质作用

触发条件：
- 片麻岩、变质岩、岩浆侵入、构造挤压。

因果链：
- 岩浆侵入 -> 周围岩石受高温影响 -> 矿物成分和结构改变 -> 形成变质岩。
- 构造挤压 -> 岩石受高压作用 -> 发生变质。

天津风格表达：
- 岩层受岩浆活动产生的高温影响，原有岩石矿物成分和结构发生改变，形成变质岩。

## 外力地貌

触发条件：
- 流水侵蚀、流水堆积、风力作用、海浪作用。

因果链：
- 地势落差大 -> 流水下切侵蚀强 -> 形成峡谷。
- 河流流速减慢 -> 泥沙沉积 -> 形成冲积平原或三角洲。
- 干旱多风 -> 风力搬运和沉积 -> 形成沙丘等风积地貌。

## 滑坡、泥石流等地质灾害

触发条件：
- 山地、峡谷、坡度大、地势起伏大。
- 降水量大、降水集中、暴雨多发。
- 地表多碎屑物、地表岩土破碎、物质疏松。
- 多喀斯特地貌、断裂发育、岩体不稳定。

因果链：
- 山体坡度大或地势起伏大 -> 岩体稳定性差 -> 易发生滑坡。
- 降水量较多且集中 -> 地表易被侵蚀，坡体含水量增加 -> 岩土体稳定性降低。
- 地表物质疏松或多碎屑物 -> 为泥石流提供物质来源 -> 易在暴雨冲刷下形成泥石流。
- 岩土破碎、岩体不稳定 -> 在重力和降水作用下易滑落、崩塌。

天津风格表达：
- 山体坡度大，岩体不稳定；降水量较多且集中，地表易被侵蚀；地表物质疏松或多碎屑物，岩体易滑落，易发生滑坡、泥石流等地质灾害。

易错点：
- 只写坡度大和降水多，漏写地表碎屑物丰富或岩土破碎这一物质基础。
- 只写“地质灾害多发”，没有写出地形、降水、松散物质之间的因果关系。
