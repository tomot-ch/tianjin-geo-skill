# 农业因果链库

## 农业区位条件

触发条件：
- 光照充足、热量充足、昼夜温差大。
- 降水适中、水源充足、土壤肥沃、地形平坦。
- 市场广阔、交通便利、技术支持、政策扶持。

因果链：
- 光照充足 -> 光合作用强 -> 有机质积累多 -> 农产品品质较好。
- 昼夜温差大 -> 夜间呼吸消耗少 -> 糖分积累多 -> 瓜果品质好。
- 地形平坦 -> 便于机械化耕作 -> 生产效率提高。
- 交通便利 -> 农产品外运方便 -> 扩大销售市场。

天津风格表达：
- 光照充足、昼夜温差大，有利于有机质积累，提高农产品品质。
- 交通便利、市场广阔，有利于农产品销售和农业增收。

## 山地牧场季节利用

触发条件：
- 山地海拔差异明显。
- 草场返青、积雪消融和气温随海拔变化。

因果链：
- 海拔升高 -> 气温降低 -> 草场生长期推迟 -> 高海拔草场多在夏季利用。
- 低海拔地区冬季气温相对较高 -> 积雪较少或融化较早 -> 适合作冬季牧场。

天津风格表达：
- 山地不同海拔地带水热条件和草场生长期不同，牧民根据季节变化进行垂直转场利用草场。

## 农业可持续发展

触发条件：
- 水资源紧张、土地退化、盐碱化、生态脆弱。

因果链：
- 水资源紧张 -> 推广节水灌溉 -> 提高水资源利用效率 -> 缓解农业用水压力。
- 生态退化 -> 调整农业结构 -> 发展生态农业和特色农业 -> 促进农业可持续发展。
