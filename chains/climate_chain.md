# 气候因果链库

## 水热条件差异

触发条件：
- 纬度、海陆位置、大气环流、地形、洋流。

因果链：
- 纬度较低 -> 太阳辐射较强 -> 热量条件较好。
- 距海较近且受湿润气流影响 -> 水汽来源充足 -> 降水较多。
- 受寒流影响 -> 降温减湿 -> 沿岸气候较干燥。
- 迎风坡 -> 气流抬升 -> 降水较多；背风坡 -> 下沉增温 -> 降水较少。

天津风格表达：
- 东岸受来自海洋的湿润气流和暖流影响，水热条件较好；西岸受副热带高压和寒流影响，降水较少。

## 锋面天气

触发条件：
- 风向转变、冷空气南下、气温下降、气压升高。

因果链：
- 冷空气南下 -> 暖空气被迫抬升 -> 冷锋过境 -> 大风、降水、降温。
- 冷锋过境后 -> 受冷气团控制 -> 气温降低、气压升高、天气转晴。

天津风格表达：
- 该地风向由偏南风转为偏北风，说明冷空气南下，冷锋过境，易出现大风、降水和降温天气。

## 气候变化与生态

触发条件：
- 气温升高、降水变化、植被覆盖变化。

因果链：
- 气温升高和降水变化 -> 水热组合改变 -> 植被生长条件变化 -> 植被覆盖度变化。
- 植被覆盖下降 -> 水土保持能力减弱 -> 生境退化 -> 生物多样性下降 -> 生态系统稳定性减弱。
