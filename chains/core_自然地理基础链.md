# 自然地理基础链

## 1. 生命存在条件链

```text
太阳稳定 → 光热稳定 → 地表温度条件稳定 → 有利于生命存在
```

```text
日地距离适中 → 温度适宜 → 液态水存在 → 有利于生命活动
```

```text
地球质量体积适中 → 引力适中 → 保持大气 → 形成适宜大气环境
```

## 2. 太阳影响链

```text
太阳辐射 → 地表获得能量 → 水循环、大气运动、生物活动和外力作用进行
```

```text
耀斑爆发 → 电离层扰动 → 无线电短波通信受影响
```

```text
太阳风增强 → 地球磁场扰动 → 磁暴 / 极光
```

## 3. 植被生态功能链

```text
森林覆盖率高 → 截留降水、增加下渗 → 涵养水源 → 调节径流
```

```text
植被根系发达 → 固结土壤 → 减弱流水侵蚀 → 保持水土
```

```text
植被覆盖率高 → 提供栖息地 → 物种丰富 → 维护生物多样性
```

```text
植被覆盖率低 → 地表裸露 → 雨滴溅蚀和径流冲刷增强 → 水土流失加剧
```

## 4. 地质过程链

```text
地壳抬升 → 地势起伏增大 → 河流落差增大 → 流水侵蚀增强
```

```text
岩层受挤压 → 发生弯曲变形 → 形成褶皱构造
```

```text
岩浆侵入 → 周围岩石受热变质 → 形成变质岩
```

```text
外力风化 → 岩石破碎 → 搬运沉积 → 形成沉积物或沉积岩
```

## 5. 水土流失链

```text
坡度大 → 地表径流流速快 → 冲刷能力强 → 水土流失严重
```

```text
降水集中且多暴雨 → 短时间径流量大 → 流水侵蚀强 → 水土流失严重
```

```text
土质疏松 → 抗侵蚀能力弱 → 易被流水侵蚀 → 水土流失严重
```

```text
植被覆盖率低 → 拦截降水和固结土壤能力弱 → 水土流失加剧
```
