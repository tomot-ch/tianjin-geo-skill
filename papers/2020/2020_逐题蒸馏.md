# 2020 天津高考地理逐题蒸馏

# 第 1—3 题：中新天津生态城、盐碱地与水循环

## 一、题目核心

材料关键词：
- 中新天津生态城
- 盐碱荒滩
- 动漫产业园区
- 智能公交站
- 南堤滨海步道公园
- 太阳能发电设施
- 永定新河
- 林地分布在堆土高地

设问关键词：
- 发展特色
- 湖泊对水循环的影响
- 林地分布在高地的原因

## 二、考点归类

- 一级考点：区域地理/人地协调
- 二级考点：生态城、低碳环保、智慧城市、水循环、盐碱化
- 核心能力：景观信息识别、区域环境问题判断

## 三、解题链

动漫产业园区
→ 高新产业

智能公交站
→ 智慧城市

步道公园
→ 生态宜居

太阳能发电设施
→ 低碳环保

园内湖泊
→ 汛期调蓄洪水
→ 调节地表径流量

盐碱荒滩
→ 低地盐分易集聚
→ 高地盐碱化程度低
→ 有利于林木成活

## 四、常见陷阱

1. 把智能公交误判为低碳环保，而忽略其“智慧城市”属性。
2. 把湖泊作用误写成减少下渗或减少蒸发。
3. 只从热量或风沙解释高地林地，忽略盐碱化。

## 五、可迁移规则

生态城题目要从产业、交通、生态、能源等功能识别景观。  
湖泊湿地在城市水循环中的常见作用是调节径流、蓄洪、补给地下水和改善生态。  
盐碱地植被布局要优先考虑盐分集聚与排盐条件。

## 六、答案话术

湖泊可在汛期分流洪水，调节地表径流，减轻洪涝压力。  
高地盐碱化程度较低，盐分不易在地表集聚，有利于林木成活。

# 第 4—5 题：粤北丹霞地貌

## 一、题目核心

材料关键词：
- 粤北地区
- 地貌景观
- 特定地质条件
- 岩层明显
- 陡崖、峰丛

设问关键词：
- 岩石类型
- 地质构造
- 外力作用
- 岩层出露地表
- 最直接内力作用

## 二、考点归类

- 一级考点：自然地理
- 二级考点：丹霞地貌、沉积岩、节理裂隙、流水侵蚀、地壳抬升
- 核心能力：地貌成因链推理

## 三、解题链

层理结构明显
→ 沉积岩

陡崖发育
→ 垂直裂隙明显

粤北湿润地区
→ 流水侵蚀强
→ 风力侵蚀弱

沉积岩原本形成于低地
→ 地壳垂直抬升
→ 出露地表
→ 流水侵蚀塑造丹霞景观

## 四、常见陷阱

1. 把丹霞地貌误判为岩浆岩或变质岩。
2. 只记“风化剥蚀”，没有明确流水侵蚀。
3. 把出露地表的内力作用误判为水平挤压。

## 五、可迁移规则

丹霞地貌常见关键词是红层、砂砾岩、层理、陡崖、垂直节理、流水侵蚀、地壳抬升。  
湿润地区地貌外力一般优先考虑流水作用。

## 六、答案话术

该地岩层层理明显，为沉积岩；受垂直裂隙控制，在流水侵蚀作用下形成陡崖和峰丛景观。  
地壳垂直抬升使沉积岩层出露地表，为外力侵蚀塑造地貌提供条件。

# 第 6—7 题：服装企业回流、产业转移与劳动力流动

## 一、题目核心

材料关键词：
- M 镇服装企业
- 迁至省会 W 市
- 后又迁回 M 镇
- 劳动力随之迁移
- 第一波回流
- 第二波回流
- 政策吸引
- 产业转移

设问关键词：
- 回流原因
- 主导区位因素

## 二、考点归类

- 一级考点：人文地理
- 二级考点：产业转移、企业回流、人口迁移、劳动力回流
- 核心能力：流程图判读、主导因素判断

## 三、解题链

企业最初迁往 W 市
→ 看重省会市场条件
→ 伴随劳动力迁入

W 市产业结构调整
→ 产业转移
→ M 镇承接产业
→ 企业第二波回流

企业回流
→ 增加就业机会
→ 改善生产条件
→ 劳动力主动回流

政策吸引
→ 促进企业迁回 M 镇

## 四、常见陷阱

1. 机械认为服装企业一定由劳动力主导。
2. 不区分第一波回流和第二波回流。
3. 把劳动力主动回流和被动回流混淆。

## 五、可迁移规则

企业迁移题要抓“迁出地推力”和“迁入地拉力”。  
服装企业虽多为劳动密集型，但不同阶段主导因素可能不同。

## 六、答案话术

M 镇承接产业转移，企业回流增加就业机会，改善生产条件，带动劳动力主动回流。  
政策吸引和生产成本优势共同促进企业迁回。

# 第 8—9 题：跨省人口流动与广东人口吸引力

## 一、题目核心

材料关键词：
- 2000 和 2010 年
- 跨省流入人口
- 广东、浙江、上海、江苏、北京
- 东部经济地带

设问关键词：
- 正确判断
- 最主要原因

## 二、考点归类

- 一级考点：人文地理
- 二级考点：人口迁移、区域经济差异、就业机会
- 核心能力：表格比较、主导因素判断

## 三、解题链

东部省份流入人口普遍增加
→ 东部流入人口规模增大

广东流入人口最多
→ 经济活力强
→ 就业机会多
→ 收入预期高
→ 吸引人口迁入

## 四、常见陷阱

1. 只看排名，不看数量变化。
2. 把交通、生态环境、政策误判为最主要原因。
3. 不区分东部、中部、西部整体格局。

## 五、可迁移规则

人口流入主因通常是经济发展水平、就业机会、收入水平。  
表格题要同时看排名和数值变化。

## 六、答案话术

广东经济活力强，就业机会多，对外来人口吸引力强。

# 第 10—11 题：城市公共服务设施与城市形态

## 一、题目核心

材料关键词：
- 西北地区某城市
- 交通、金融、医疗、文化设施分布密度
- 虚线圈
- 中心城区形态

设问关键词：
- 最可能城市功能
- 主要影响因素

## 二、考点归类

- 一级考点：人文地理
- 二级考点：城市功能区、公共服务设施、城市空间形态
- 核心能力：多图叠加判读

## 三、解题链

交通设施密度大
→ 通达性好

金融设施密度大
→ 商业服务功能强

综合判断
→ 商业和住宅区可能性大

中心城区呈条带状
→ 西北地区地形制约明显
→ 主要因素是地形

## 四、常见陷阱

1. 看到交通设施多就只选交通因素。
2. 忽略中心城区整体呈条带状。
3. 把工业和仓储物流误放到中心城区。

## 五、可迁移规则

城市功能区判断要看设施组合。  
城市形态狭长、条带状，常与山谷、河谷、地形限制有关。

## 六、答案话术

交通和金融服务设施密度较大，说明该区域交通便利、商业服务功能较强，适宜形成商业和住宅功能区。  
中心城区呈条带状分布，主要受地形条件制约。

# 第 12—13 题：天气形势图与风向判断

## 一、题目核心

材料关键词：
- 某日 14 时平面气压场
- 等压线
- 华北平原
- 高压脊
- 城市风向

设问关键词：
- 天气状况
- 风向偏南

## 二、考点归类

- 一级考点：自然地理
- 二级考点：高压脊、等压线、风向、天气状况
- 核心能力：天气图判读

## 三、解题链

华北平原
→ 等压线稀疏
→ 风力较小

位于高压脊附近
→ 下沉气流
→ 天气晴朗
→ 风和日丽

风向判断
→ 水平气压梯度力由高压指向低压
→ 北半球向右偏转
→ 判断福州为偏南风

## 四、常见陷阱

1. 只看气压值，不看高压脊/低压槽。
2. 忘记北半球近地面风向向右偏。
3. 把等压线稀疏区误判为狂风暴雨。

## 五、可迁移规则

高压脊附近通常天气晴朗。  
风向判断必须先画水平气压梯度力，再考虑地转偏向力和摩擦力。

## 六、答案话术

高压脊控制下盛行下沉气流，不易成云致雨，天气晴朗。  
等压线较稀疏，水平气压梯度力较小，风力较弱。

# 第 14—15 题：住宅小区日影与地球运动

## 一、题目核心

材料关键词：
- 住宅小区
- 二至日
- 8:00 和 16:00 日影
- 休闲广场
- 楼影遮挡

设问关键词：
- 最可能位于
- 遮挡面积最大

## 二、考点归类

- 一级考点：自然地理
- 二级考点：地球运动、太阳高度、日影方向与长度
- 核心能力：图像方位判断

## 三、解题链

太阳高度越低
→ 影子越长

冬至日
→ 北半球太阳高度较低
→ 楼影长
→ 遮挡面积大

上午 8:00
→ 太阳在东南方向
→ 影子指向西北
→ 结合广场位置判断遮挡最大

## 四、常见陷阱

1. 只看夏至昼长，不看太阳高度。
2. 把影子方向和太阳方向混淆。
3. 不结合广场位置判断遮挡面积。

## 五、可迁移规则

楼影问题要同时看太阳高度、太阳方位、建筑与目标区相对位置。  
冬至日太阳高度低，影子通常更长。

## 六、答案话术

冬至日太阳高度较低，住宅楼影较长，若楼影方向指向休闲广场，则遮挡面积较大。

# 第 16 题：寿光蔬菜、互联网销售与日照绿茶

## 一、题目核心

材料关键词：
- 山东寿光
- 现代农业科技
- 温室蔬菜
- 政府扶持
- 疫情期间供应蔬菜
- 临沂小商品集散中心
- 线上销售
- 日照绿茶
- 南茶北引
- 叶片厚、香味浓、耐冲泡

设问关键词：
- 有利条件
- 互联网销售好处
- 从气候角度分析原因

## 二、考点归类

- 一级考点：人文地理/农业地理
- 二级考点：农业区位、电子商务、特色农产品品质
- 核心能力：区位分析、影响分析、气候品质链推理

## 三、解题链

寿光
→ 平原地形
→ 土地资源丰富
→ 蔬菜基地规模大
→ 温室技术
→ 全年生产
→ 交通便利
→ 政策扶持

互联网销售
→ 扩大市场
→ 降低成本
→ 满足个性化需求
→ 减少直接接触

日照
→ 光照较好
→ 昼夜温差大
→ 积温低
→ 茶树生长慢
→ 有益物质积累
→ 叶厚、香浓、耐冲泡

## 四、常见陷阱

1. 只写自然条件，忽略温室技术和政策扶持。
2. 互联网销售只写“方便”，缺乏市场、成本、需求、防疫角度。
3. 茶叶品质只写降水，不写光照、昼夜温差和积温。

## 五、可迁移规则

农业区位题按照土地、气候、水源、技术、交通、市场、政策组织。  
农产品品质题常从光照、昼夜温差、热量、生长期、有机质积累解释。

## 六、答案话术

温室技术改善热量条件，可保障蔬菜全年生产。  
互联网销售可扩大市场范围，降低销售成本，提高经济效益。  
光照条件较好、昼夜温差较大，有利于有益物质积累。

# 第 17 题：贵州旅游资源、气温差异与泥石流

## 一、题目核心

材料关键词：
- 贵州
- 岩溶地貌广布
- 世界自然遗产
- 山区面积广大
- 兴义、铜仁
- 川滇黔交界
- 泥石流

设问关键词：
- 评价旅游资源开发条件
- 气温年较差
- 冬季气温高、夏季气温低
- 泥石流自然条件

## 二、考点归类

- 一级考点：区域地理/自然地理
- 二级考点：旅游资源评价、气温影响因素、泥石流灾害
- 核心能力：图表读取、自然原因分析

## 三、解题链

世界自然遗产
→ 知名度高
→ 特色突出
→ 游览价值高

山区面积广
→ 地表崎岖
→ 交通通达性差

兴义冬季纬度低
→ 正午太阳高度大
→ 昼长较长
→ 冬季风影响小
→ 冬季气温高

兴义海拔高
→ 夏季气温低

泥石流
→ 山地坡度大
→ 降水量大
→ 碎屑物丰富
→ 易发生

## 四、常见陷阱

1. 只写旅游资源价值，不写交通通达性。
2. 比较气温时只看纬度，不看海拔和季风影响。
3. 泥石流条件只写降水，不写地形和松散物质。

## 五、可迁移规则

旅游开发评价要分有利和不利。  
气温差异题要综合纬度、海拔、坡向、季风影响。  
泥石流条件按地形、降水、松散物质三要素组织。

## 六、答案话术

拥有世界自然遗产，旅游资源知名度高、特色突出，游览价值高。  
山区面积广大，地表崎岖，交通通达性较差。  
山地坡度较大，年降水量较大，地表碎屑物丰富，易发生泥石流。

# 第 18 题：伊朗铁路、中伊渔业合作与内流湖盐度

## 一、题目核心

材料关键词：
- 伊朗
- 丝绸之路经济带
- 21 世纪海上丝绸之路
- 铁路网络建设和改造
- 中伊渔业合作
- 甲湖淡水湖
- 乙湖盐湖
- 气候干旱、蒸发量大
- 内流湖

设问关键词：
- 对经济发展的作用
- 各自优势条件
- 淡水湖和盐湖原因

## 二、考点归类

- 一级考点：区域地理/人文地理
- 二级考点：交通建设、国际合作、水文盐度
- 核心能力：意义分析、分主体作答、水文过程推理

## 三、解题链

铁路建设
→ 加强对外联系
→ 完善交通网络
→ 提高运力
→ 带动相关产业
→ 增加就业

中伊渔业合作
→ 伊朗渔业资源丰富
→ 中国市场广阔、资金技术充足
→ 互利共赢

甲湖
→ 多河流汇入
→ 有河道流向乙湖
→ 盐分被带走
→ 淡水湖

乙湖
→ 只有河流汇入
→ 无河流流出
→ 蒸发强烈
→ 盐分积累
→ 盐湖

## 四、常见陷阱

1. 交通意义只写“方便出行”，没有写经济带动作用。
2. 国际合作不分主体。
3. 内流湖盐度只写蒸发，不看是否有出流。

## 五、可迁移规则

交通建设意义按联系、网络、运力、产业、就业组织。  
国际合作题必须分双方写资源、市场、资金、技术等优势。  
湖泊盐度看水量收入、排盐通道和蒸发强弱。

## 六、答案话术

铁路建设可加强对外运输联系，完善本国交通网络，提高客货运输能力，带动相关产业发展。  
伊朗渔业资源丰富；中国市场广阔，资金充足，技术先进。  
甲湖有河流流出，可带走盐分；乙湖无河流流出，蒸发强烈，盐分不断积累。
