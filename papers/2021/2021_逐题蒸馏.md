# 2021 天津高考地理逐题蒸馏

# 第 1 题：天津蓟州地层与沧海桑田

## 一、题目核心

材料关键词：
- 天津蓟州北部山区
- 景观照片
- 中上元古界地层
- 沧海桑田

设问关键词：
- 能记录地球沧海桑田变化

## 二、考点归类

- 一级考点：自然地理/区域地理
- 二级考点：地层、化石、海陆变迁
- 核心能力：景观信息识别

## 三、解题链

沧海桑田
→ 海陆变迁
→ 地层中可能保存海洋生物化石
→ 可证明当地曾为海洋环境
→ 中上元古界地层最符合

## 四、常见陷阱

1. 把湖泊、水域形成误判为沧海桑田。
2. 把长城、森林等景观误认为地质变迁证据。
3. 不理解“沧海桑田”对应海陆变迁。

## 五、可迁移规则

反映海陆变迁的材料常见关键词：地层、海洋化石、石灰岩、贝壳层、沉积岩。

## 六、答案话术

中上元古界地层可能保存海洋生物化石，可反映当地曾发生海陆变迁。

# 第 2—3 题：华北地质剖面与岩溶地貌

## 一、题目核心

材料关键词：
- 华北某地
- 地质剖面
- 砂岩、页岩、石灰岩
- 断层
- 甲地岩溶地貌
- 与西南同类岩层比较

设问关键词：
- 地层由老到新
- 岩溶地貌发育程度较低的原因

## 二、考点归类

- 一级考点：自然地理
- 二级考点：沉积岩层序、断层、岩溶地貌、水热条件
- 核心能力：剖面图判读、比较成因分析

## 三、解题链

沉积岩
→ 一般下老上新
→ 结合断层和覆盖关系
→ 判断地层新老

岩溶地貌
→ 可溶性岩石受流水溶蚀
→ 与西南同类岩层比较
→ 岩石可溶性差异小
→ 华北水热条件较差
→ 溶蚀作用弱
→ 岩溶发育程度低

## 四、常见陷阱

1. 只按绝对高度判断岩层新老。
2. 忽略断层造成的地层错动。
3. 同类岩层比较时仍然选“岩石可溶性低”。

## 五、可迁移规则

沉积岩地层通常下老上新，但要结合断层、侵入关系和覆盖关系判断。  
同类可溶岩层比较岩溶发育强弱，优先看水热条件。

## 六、答案话术

同类岩层条件下，华北地区水热条件较差，流水溶蚀作用较弱，岩溶地貌发育程度较低。

# 第 4—5 题：北方沙尘暴与冷锋南下

## 一、题目核心

材料关键词：
- 2021 年 3 月中旬
- 北方大规模沙尘暴
- 源于蒙古国
- 天气形势图
- 沙尘天气分布

设问关键词：
- 分布与天气形势吻合
- 推动沙尘暴快速移动的主要原因

## 二、考点归类

- 一级考点：自然地理
- 二级考点：天气系统、冷锋、沙尘暴、等压线
- 核心能力：天气图和动态过程匹配

## 三、解题链

蒙古国沙源
→ 春季地表干燥
→ 冷空气快速南下
→ 冷锋形成
→ 大风卷起沙尘
→ 沙尘向东南方向移动
→ 对应图中沙尘带变化

## 四、常见陷阱

1. 把沙尘暴推动原因误判为气旋西移或反气旋东进。
2. 只看沙尘范围，不看移动方向。
3. 忽略冷锋带来的大风作用。

## 五、可迁移规则

春季北方沙尘天气常与冷锋、大风、干燥下垫面、沙源地有关。  
判断沙尘路径要结合风向、锋面移动方向和沙源位置。

## 六、答案话术

冷锋快速南下带来大风天气，卷起沙尘并向东南方向移动，形成沙尘天气。

# 第 6—7 题：川藏铁路、雪线林线与雪豹活动范围

## 一、题目核心

材料关键词：
- 川藏铁路
- 成都至拉萨
- 林芝附近山地
- 雪豹活动
- 雪线之下、林线之上
- 迎风坡、背风坡

设问关键词：
- 川藏铁路沿线特征
- 迎风坡活动范围小的原因

## 二、考点归类

- 一级考点：自然地理/区域地理
- 二级考点：青藏高原、垂直自然带、雪线、林线、坡向
- 核心能力：区域差异判断、垂直带谱推理

## 三、解题链

川藏铁路
→ 穿越四川盆地和青藏高原
→ 地势高差大
→ 水热差异明显
→ 气温和干湿状况差异大

迎风坡
→ 降水多
→ 雪线低
→ 水热条件较好
→ 林线高
→ 雪线与林线之间范围小
→ 雪豹活动范围小

## 四、常见陷阱

1. 认为川藏铁路全线都位于第一级阶梯。
2. 把迎风坡简单理解为雪线和林线都低。
3. 忘记雪豹活动范围是“雪线以下、林线以上”的夹层。

## 五、可迁移规则

雪线受气温和降水共同影响；林线主要受热量和水分影响。  
动物垂直活动范围题要先明确物种所在高度带，再比较上下限。

## 六、答案话术

迎风坡降水较多，雪线较低，林线较高，雪线与林线之间的空间较小。

# 第 8 题：农民工流动与产业转移

## 一、题目核心

材料关键词：
- 2010—2019 年
- 全国农民工输入地地区分布变化
- 东部、中部、西部
- 产业结构调整

设问关键词：
- 流动特点
- 推断原因

## 二、考点归类

- 一级考点：人文地理
- 二级考点：人口迁移、产业转移、劳动密集型产业
- 核心能力：折线图趋势判读

## 三、解题链

图示中西部比例上升
→ 东部比例下降
→ 产业结构调整
→ 劳动密集型产业向中西部转移
→ 中西部就业机会增加
→ 农民工流入比例上升

## 四、常见陷阱

## 六、答案话术

# 第 14—15 题：北极圈极昼、太阳高度与经度定位

## 一、题目核心

材料关键词：
- 北极圈内某地
- 北京时间 6 月 22 日 16 时
- 一天中太阳高度最小值为 10°
- 夏至日
- 极昼

设问关键词：
- 纬度
- 位于哪里

## 二、考点归类

- 一级考点：自然地理
- 二级考点：地球运动、极昼、太阳高度、地方时、经度
- 核心能力：太阳高度计算、时差定位

## 三、解题链

6 月 22 日
→ 太阳直射北回归线
→ 北极圈及其以北极昼

极昼区最小太阳高度
→ 地方时 0 点太阳高度

北极圈 66°34′N 最小太阳高度为 0°
→ 该地最小太阳高度 10°
→ 纬度 = 66°34′N + 10°
→ 76°34′N

北京时间 16 时
→ 当地地方时 0 时
→ 相差 8 小时
→ 经度约 120°W
→ 北美北部

## 四、常见陷阱

1. 把最小太阳高度当成正午太阳高度。
2. 忘记极昼区最小太阳高度出现在地方时 0 点。
3. 经度换算方向搞反。

## 五、可迁移规则

极昼最小太阳高度可以用于反推纬度。  
北京时间与地方时差可用于反推经度。

## 六、答案话术

极昼区一天中最小太阳高度出现在地方时 0 点；该地最小太阳高度比北极圈高 10°，故纬度为 76°34′N。

# 第 16 题：京津冀协同、产业承接、交通一体化与运河文化

材料关键词：
- 京津冀协同发展
- 交通一体化
- 生态环境保护
- 产业升级转移
- 北京非首都功能转移
- 天津武清区
- 轻轨客运铁路
- 运河文化带

设问关键词：
- 承接先进制造业优势
- 交通一体化意义
- 保护性开发运河文化遗产措施

## 二、考点归类

- 一级考点：区域地理/人文地理
- 二级考点：产业承接、交通网络、文化遗产保护
- 核心能力：区位分析、意义分析、措施表达

## 三、解题链

武清区
→ 距北京近
→ 高速公路、铁路通过
→ 交通便利
→ 生产成本较低
→ 工业基础好
→ 政策支持
→ 承接先进制造业

轻轨铁路
→ 连接京津地铁
→ 联程联运
→ 完善交通网络
→ 增加轨道覆盖
→ 节省出行时间

运河文化
→ 挖掘历史民俗
→ 突出天津地域特色
→ 整治沿河景观
→ 保护生态环境
→ 发展旅游相关产业

## 四、常见陷阱

1. 产业承接只写距离近，不写交通、成本、工业基础和政策。
2. 交通一体化只写“方便出行”，缺少网络和联程联运。
3. 文化遗产开发只写旅游，不写保护。

## 五、可迁移规则

产业承接区位按照距离、交通、成本、产业基础、政策组织。  
文化遗产保护性开发要坚持保护优先、特色突出、产业带动。

## 六、答案话术

武清区距北京近，交通便利，生产成本较低，工业基础较好，有利于承接北京先进制造业。  
轻轨铁路建设有利于推动不同运输方式联程联运，完善京津冀交通网络，节省出行时间。  
保护性开发运河文化遗产，应挖掘历史民俗文化，突出地域特色，整治沿河景观，发展相关旅游产业。

# 第 17 题：甘南红色旅游、旅游扶贫与特色畜牧业

## 一、题目核心

材料关键词：
- 甘青川交界
- 河流、湖沼、雪山、草甸、藏寨、寺院
- 红军长征纪念地
- 甘南藏区
- 天津对口扶贫
- 基础设施和商业网点
- 藏式民舍、度假村
- 放养牦牛、藏绵羊

设问关键词：
- 红色旅游资源功能和价值
- 旅游扶贫影响
- 特色畜牧业自然条件

## 二、考点归类

- 一级考点：人文地理/区域地理
- 二级考点：红色旅游、旅游扶贫、特色畜牧业
- 核心能力：影响分析、农业自然条件分析

## 三、解题链

红军长征纪念地
→ 红色旅游资源
→ 革命传统教育功能
→ 历史文化价值高

天津援建
→ 基础设施改善
→ 商业网点增加
→ 接待能力提高
→ 旅游资源开发保护
→ 知名度提高
→ 客源市场扩大

草甸地区
→ 海拔高
→ 气候冷湿
→ 草甸面积大
→ 水源充足
→ 适合牦牛、藏绵羊放养

## 四、常见陷阱

1. 红色旅游只写经济价值，不写教育功能。
2. 旅游扶贫影响只写收入增加，不写接待能力和资源开发保护。

红色旅游资源价值要写革命传统教育功能和历史文化价值。

畜牧业自然条件看草场、水源、气候、海拔。

## 六、答案话术

红色旅游资源具有突出的革命传统教育功能和较高历史文化价值。  
天津援建基础设施和商业网点，可改善旅游服务设施，提高接待能力，扩大客源市场。  
当地海拔较高，草甸面积大，水源充足，适合发展以牦牛、藏绵羊为主的特色畜牧业。

# 第 18 题：北极块状冰、生态脆弱性与温室效应

- 北大西洋暖流
- 斯瓦尔巴群岛
- 生态环境脆弱
- 冰川、冻土、植被变化

设问关键词：
- 分布特征原因
- 生态脆弱原因
- 温室效应增强或减弱过程

## 二、考点归类

- 一级考点：自然地理
- 二级考点：北极海冰、洋流、生态脆弱性、全球变暖、温室效应
- 核心能力：自然过程链推理

## 三、解题链

极点附近
→ 纬度高
→ 气温低
→ 块状冰分布多

欧洲以北海区
→ 受北大西洋暖流影响
→ 海水增温
→ 与大西洋温暖海水交换多
→ 块状冰分布少

斯瓦尔巴群岛
→ 纬度高
→ 气温低
→ 环境极端
→ 生物种类少
→ 生长缓慢
→ 岛屿面积小、相对封闭
→ 生态破坏后难恢复

全球变暖
→ 冰川减少
→ 地表反射率下降
→ 吸收太阳辐射增加
→ 温室效应增强

冻土融化
→ 释放温室气体
→ 温室效应增强

植被增加
→ 光合作用吸收二氧化碳
→ 温室效应减弱

## 四、常见陷阱

1. 只用纬度解释欧洲以北海区，忽略北大西洋暖流。
2. 生态脆弱只写气候寒冷，不写物种少、生长慢、封闭性。
3. 温室效应题不写过程链。

## 五、可迁移规则

北极海冰分布要看纬度、洋流、海陆通道和海水交换。  
生态脆弱性从环境极端、物种少、生长慢、系统封闭解释。  
温室效应变化要写碳循环和地表辐射收支过程。

## 六、答案话术

欧洲以北海区受北大西洋暖流影响，海水增温，永久性块状冰分布面积较小。  
该群岛气温低，环境极端，生物数量和种类少，生长缓慢，一旦破坏很难恢复。  
冰川面积减小，地表吸收太阳辐射增加，地面辐射增强，可能导致温室效应增强。  
植被覆盖率增加，光合作用吸收更多二氧化碳，可能导致温室效应减弱。
