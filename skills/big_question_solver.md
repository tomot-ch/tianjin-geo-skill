# 综合题求解器

本文件用于处理天津地理大题。大题不是背模板，而是把材料中的条件转化为可以得分的推理链。

## 1. 总流程

1. 看设问：判断问原因、影响、措施、比较还是判断理由。
2. 定专题：河流、气候、农业、工业、城市、交通、地貌、生态。
3. 抽材料：圈出位置、时间、地形、气候、水源、资源、交通、市场、政策等条件。
4. 转术语：把材料语言转成规范地理表达。
5. 串链条：写出“条件 → 过程 → 结果”。
6. 组答案：按得分点分句，避免重复。
7. 查遗漏：自然条件和社会经济条件是否按设问要求取舍。

## 2. 原因题

答题结构：
- 自然原因：位置、气候、地形、水源、土壤、植被。
- 人文原因：资源、交通、市场、劳动力、科技、政策、产业基础。

生成方式：
- 材料条件 + 地理过程 + 结果。

示例：
- 纬度较低，热量充足，有利于农作物生长。
- 交通便利，有利于原料输入和产品输出。

## 3. 影响题

答题结构：
- 经济影响：促进区域经济发展、增加收入、带动相关产业。
- 社会影响：增加就业、完善基础设施、改善居民生活。
- 生态影响：改善生态环境，或加剧污染、破坏生物多样性。

生成方式：
- 先写直接影响，再写进一步影响。

示例：
- 产业发展可增加就业机会，提高居民收入，促进区域经济发展。

## 4. 措施题

答题结构：
- 问题是什么，就对应解决什么。
- 自然问题配生态治理，产业问题配结构优化，资源问题配提高利用效率。

常用表达：
- 加强生态保护，恢复植被。
- 优化产业结构，延长产业链。
- 提高资源利用效率，发展循环经济。
- 完善交通基础设施，增强区域联系。
- 因地制宜发展特色产业。

## 5. 比较题

答题结构：
- A 地：条件 + 结果。
- B 地：条件 + 结果。
- 最后指出差异。

注意：
- 不要只写 A 地。
- 不要只写结论，如“A 比 B 好”。
- 差异要来自材料。

## 6. 判断理由题

答题结构：
- 先判断。
- 再列材料依据。
- 最后说明地理过程。

示例：
- 判断该河流以雨水补给为主。理由是流量峰值出现在夏季，与当地夏季降水集中一致，说明河流补给受降水影响明显。

## 7. 大题检查清单

- 是否回答了设问动词？
- 是否使用了材料条件？
- 是否有完整因果链？
- 是否用了规范术语？
- 是否避免重复得分点？
- 是否区分自然条件和社会经济条件？
- 是否有材料无法支持的硬扩展？
