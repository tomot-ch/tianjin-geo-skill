# 条件识别系统

本文件用于把题目材料中的“显性条件”转化为“地理含义”。使用时不要直接背结论，而要按“条件 → 自动联想 → 设问方向 → 标准表达 → 因果链”的顺序推理。

## 1. 纬度条件

### 条件：纬度高

自动联想：
- 热量不足
- 气温较低
- 生长期较短
- 蒸发较弱
- 河流可能有结冰期

常见设问：
- 分析农业发展的限制条件。
- 分析河流结冰期的原因。
- 分析作物熟制较低的原因。

标准表达：
- 纬度较高，气温较低，热量条件不足，作物生长期较短。

可生成因果链：
- 纬度高 → 太阳高度角较小 → 获得太阳辐射少 → 气温低 → 热量不足

### 条件：纬度低

自动联想：
- 热量充足
- 气温较高
- 生长期较长
- 作物熟制较高
- 蒸发可能较强

常见设问：
- 分析农业发展的有利条件。
- 分析热带经济作物分布原因。
- 分析蒸发旺盛或水资源紧张的原因。

标准表达：
- 纬度较低，热量充足，作物生长期较长，有利于农业生产。

可生成因果链：
- 纬度低 → 太阳高度角较大 → 获得太阳辐射多 → 气温高 → 热量充足

## 2. 海陆位置

### 条件：临海

自动联想：
- 海运便利
- 对外联系方便
- 受海洋调节明显
- 降水可能较多
- 港口、外向型经济条件较好

常见设问：
- 分析港口发展的区位条件。
- 分析工业或城市发展的区位优势。
- 分析气候海洋性特征。

标准表达：
- 临近海洋，海运便利，对外联系方便，有利于发展外向型经济。

可生成因果链：
- 临海 → 海运便利 → 运输成本较低 → 对外联系方便 → 促进区域经济发展

### 条件：深居内陆

自动联想：
- 距海远
- 水汽难以到达
- 降水较少
- 气候大陆性强
- 交通联系可能受限

常见设问：
- 分析干旱原因。
- 分析农业水源限制。
- 分析区域开发不利条件。

标准表达：
- 深居内陆，距海较远，水汽难以到达，降水较少。

可生成因果链：
- 深居内陆 → 海洋水汽难以到达 → 降水少 → 水资源不足

## 3. 地形

### 条件：地形平坦

自动联想：
- 建设成本较低
- 交通线路布局便利
- 耕地集中连片
- 城市扩展空间较大

常见设问：
- 分析城市建设条件。
- 分析农业发展的有利条件。
- 分析交通线布局原因。

标准表达：
- 地形平坦，便于城市建设、交通线路布局和农业机械化生产。

可生成因果链：
- 地形平坦 → 建设难度小 → 交通和城市布局便利 → 区域开发条件较好

### 条件：地势起伏大

自动联想：
- 交通建设难度大
- 耕地破碎
- 水流速度快
- 水能资源可能丰富
- 易发生滑坡、泥石流
- 若降水集中且地表多碎屑物或岩土破碎，滑坡、泥石流风险更高

常见设问：
- 分析交通建设困难。
- 分析水能资源丰富原因。
- 分析地质灾害多发原因。

标准表达：
- 地势起伏较大，交通建设难度大，但河流落差大，水能资源较丰富。
- 山体坡度大，岩体不稳定；降水集中，地表易被侵蚀；地表物质疏松或多碎屑物，易发生滑坡、泥石流。

可生成因果链：
- 地势起伏大 → 河流落差大 → 水流速度快 → 水能资源丰富
- 地势起伏大 → 坡度大、岩体不稳定 → 降水集中侵蚀 → 松散碎屑物被搬运 → 滑坡、泥石流多发

## 4. 气候

### 条件：降水丰富

自动联想：
- 水资源较丰富
- 河流补给量大
- 植被覆盖可能较好
- 洪涝风险可能较高

常见设问：
- 分析河流流量大原因。
- 分析农业有利条件。
- 分析洪涝灾害原因。

标准表达：
- 降水较多，地表径流增加，河流补给量较大。

可生成因果链：
- 降水丰富 → 地表径流增加 → 河流补给增加 → 河流流量较大

### 条件：降水集中

自动联想：
- 径流季节变化大
- 汛期明显
- 洪涝灾害风险较高
- 旱涝季节差异明显

常见设问：
- 分析河流水位季节变化大的原因。
- 分析洪涝多发原因。
- 分析农业生产不稳定原因。

标准表达：
- 降水集中于夏季，河流补给季节差异明显，径流量季节变化大。

可生成因果链：
- 降水集中 → 短时间地表径流增加 → 河流水位上涨 → 洪涝风险增加

## 5. 水源

### 条件：河流、湖泊附近

自动联想：
- 生产生活用水便利
- 灌溉条件较好
- 城市和农业发展有水源保障
- 也可能面临洪涝风险

常见设问：
- 分析农业区位条件。
- 分析城市形成原因。
- 分析工业布局条件。

标准表达：
- 临近河流，水源充足，能够满足生产生活和农业灌溉需求。

可生成因果链：
- 临近河流 → 水源充足 → 灌溉便利 → 有利于农业发展

### 条件：水资源短缺

自动联想：
- 制约农业和工业发展
- 生态环境脆弱
- 需要节水、调水和提高利用效率

常见设问：
- 分析区域发展限制因素。
- 提出缓解水资源短缺措施。
- 分析农业生产不利条件。

标准表达：
- 水资源不足，制约工农业生产和城市发展。

可生成因果链：
- 降水少或用水量大 → 水资源短缺 → 生产生活受限 → 区域发展受制约

## 6. 土壤

### 条件：土壤肥沃

自动联想：
- 农业基础好
- 作物产量较高
- 有利于种植业发展

常见设问：
- 分析农业发展的自然条件。
- 分析商品粮基地形成原因。

标准表达：
- 土壤肥沃，有利于农作物生长和提高农产品产量。

可生成因果链：
- 土壤肥沃 → 养分充足 → 作物生长条件好 → 农业产量较高

### 条件：土壤贫瘠或盐碱化

自动联想：
- 作物生长受限
- 土地生产力下降
- 需要改良土壤

常见设问：
- 分析农业发展的不利条件。
- 提出土地治理措施。

标准表达：
- 土壤肥力较低或盐碱化较重，不利于农作物生长。

可生成因果链：
- 蒸发强或排水不畅 → 盐分在地表积累 → 土壤盐碱化 → 农业生产受影响

## 7. 资源

### 条件：矿产资源丰富

自动联想：
- 原料或能源充足
- 有利于资源型工业发展
- 可能形成工业基地
- 也可能带来生态破坏

常见设问：
- 分析工业区位条件。
- 分析资源型城市兴起原因。
- 分析产业转型必要性。

标准表达：
- 矿产资源丰富，原料和能源供应充足，有利于发展相关工业。

可生成因果链：
- 矿产资源丰富 → 原料供应充足 → 生产成本较低 → 工业发展条件较好

## 8. 交通

### 条件：交通干线交会

自动联想：
- 交通便利
- 区位优势明显
- 有利于物流、人流、信息流集散
- 市场辐射范围扩大

常见设问：
- 分析城市、商业中心或物流中心形成原因。
- 分析工业布局优势。

标准表达：
- 位于交通干线交会处，交通便利，有利于人流、物流和信息流集散。

可生成因果链：
- 交通干线交会 → 运输便利 → 联系范围扩大 → 带动产业集聚

## 9. 市场

### 条件：靠近消费市场

自动联想：
- 产品销售便利
- 运输成本较低
- 信息反馈快
- 适合布局市场导向型工业或城郊农业

常见设问：
- 分析工业布局原因。
- 分析乳畜业、花卉蔬菜等农业布局原因。

标准表达：
- 靠近消费市场，产品运输距离短，市场需求量大。

可生成因果链：
- 靠近市场 → 运输成本低 → 产品销售便利 → 经济效益较高

## 10. 劳动力

### 条件：劳动力丰富

自动联想：
- 用工充足
- 劳动力成本可能较低
- 有利于劳动密集型产业发展

常见设问：
- 分析工业转移承接条件。
- 分析产业发展区位优势。

标准表达：
- 劳动力资源丰富，能够满足产业发展对用工的需求。

可生成因果链：
- 劳动力丰富 → 用工充足 → 生产成本较低 → 有利于劳动密集型产业发展

## 11. 科技

### 条件：科技水平高

自动联想：
- 创新能力强
- 生产效率高
- 产品附加值高
- 有利于高新技术产业发展

常见设问：
- 分析高新技术产业布局条件。
- 分析产业升级原因。

标准表达：
- 科技水平较高，创新能力强，有利于提高产品附加值和产业竞争力。

可生成因果链：
- 科技水平高 → 创新能力强 → 产品附加值提高 → 产业竞争力增强

## 12. 政策

### 条件：政策支持

自动联想：
- 基础设施改善
- 资金和项目投入增加
- 产业集聚加快
- 区域发展机会增多

常见设问：
- 分析新区、开发区或产业园发展原因。
- 分析区域产业转型条件。

标准表达：
- 政策支持力度较大，有利于完善基础设施、吸引投资和促进产业集聚。

可生成因果链：
- 政策支持 → 资金和项目投入增加 → 基础设施改善 → 产业发展条件优化
