# 材料提取规则

本文件用于把题目材料中的文字、图表、区域图和数据变化，提取为可推理的条件。做题时先提取材料，再调用条件识别系统和因果链。

## 1. 文字材料提取

看到“近年来产业结构不断优化”：

自动联想：
- 产业升级
- 第三产业比重提高
- 经济结构改善
- 资源依赖程度可能下降

可用于回答：
- 产业转型成效
- 区域经济发展变化
- 城市功能提升

看到“交通干线交会”：

自动联想：
- 交通便利
- 区位优势明显
- 有利于物流、人流、信息流集散
- 市场辐射范围扩大

可用于回答：
- 城市形成原因
- 商业中心区位
- 工业或物流业布局

看到“降水集中在夏季”：

自动联想：
- 河流夏季流量大
- 径流季节变化大
- 可能出现洪涝灾害
- 农业用水季节差异明显

可用于回答：
- 河流汛期
- 洪涝灾害
- 农业生产不稳定

## 2. 图表材料提取

读气温曲线：
- 最高温出现月份 → 判断南北半球和热量季节。
- 最低温是否低于 0℃ → 判断河流结冰期、作物越冬条件。
- 年温差大小 → 判断大陆性或海洋性。

读降水柱状图：
- 年降水总量 → 判断水分条件。
- 降水集中月份 → 判断雨季、汛期和洪涝风险。
- 降水季节差异 → 判断径流季节变化。

读流量过程线：
- 流量峰值月份 → 判断补给类型。
- 是否有春汛 → 联想季节性积雪融水。
- 夏季高温期是否流量大 → 联想冰雪融水或雨水补给。
- 全年是否稳定 → 联想地下水或湖泊调节。

## 3. 区域图提取

位置：
- 纬度位置 → 热量、结冰期、作物熟制。
- 海陆位置 → 降水、海运、气候海洋性。
- 相对位置 → 靠近市场、资源、港口或交通节点。

地形：
- 平原 → 地形平坦，适合城市、交通和农业布局。
- 山地 → 地势起伏大，水能可能丰富，交通建设难度大。
- 河谷 → 水源、交通通道、聚落分布。

水系：
- 支流多 → 汇水量大。
- 河道弯曲 → 排水不畅或河曲发育。
- 湖泊湿地多 → 调蓄洪水、改善生态。

## 4. 数据变化提取

看到“比重上升”：
- 说明该要素地位增强。
- 如果是第三产业比重上升，可联想产业结构优化。
- 如果是城镇人口比重上升，可联想城市化水平提高。

看到“数量增加”：
- 先判断增加对象是什么，再判断影响。
- 企业增加 → 产业集聚增强。
- 游客增加 → 旅游业发展，服务业收入增加。
- 建设用地增加 → 城市扩张，生态压力可能加大。

看到“波动大”：
- 联想季节变化大、稳定性差、受自然条件影响明显。
- 河流流量波动大 → 补给季节变化大。
- 农产品产量波动大 → 受气候灾害影响较明显。

## 5. 设问关键词提取

“分析原因”：
- 回答条件和过程。
- 格式：条件 → 地理过程 → 结果。

“说明影响”：
- 分正面和负面。
- 注意影响对象：经济、社会、生态、交通、产业、居民生活。

“提出措施”：
- 措施要对应问题原因。
- 尽量写可操作表达，如“加强”“完善”“提高”“优化”。

“比较差异”：
- 必须两边都写。
- 不要只描述一方，要指出差异来源。

“判断并说明理由”：
- 先给判断，再写材料依据。
- 理由要紧扣图文材料，不要只背知识点。
