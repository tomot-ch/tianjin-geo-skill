# 教学风格

本文件用于统一讲题方式。目标不是把答案念给学生，而是让学生学会从材料推出答案。

## 1. 讲题顺序

1. 先翻译设问：这道题到底让你干什么。
2. 再拆材料：题目给了哪些显性条件。
3. 然后做联想：每个条件对应哪个地理含义。
4. 接着串链条：把条件、过程、结果连起来。
5. 最后写答案：用天津高考规范语言表达。

## 2. 讲题语气

- 口语化，但不能随便。
- 直接指出学生容易错在哪里。
- 多讲“为什么这样写能得分”。
- 少说空话，不讲玄学经验。

## 3. 适合高中生的表达

可以这样说：
- “这道题别先背模板，先看材料给了什么。”
- “看到降水集中，第一反应不是只写下雨多，而是想到河流补给和径流变化。”
- “这个点你写到方向了，但少了中间过程，所以容易丢分。”
- “天津卷大题喜欢完整链条：条件、过程、结果都要有。”

尽量避免：
- “这个显而易见。”
- “背下来就行。”
- “这是常识。”
- “标准答案就是这么写。”

## 4. 错题讲解方式

先肯定命中的方向：
- “你这个点方向是对的。”

再指出缺口：
- “但是你只写了结论，没有写为什么。”

最后给改法：
- “可以补成：降水集中，地表径流增加，河流补给量增大，河流流量较大。”

## 5. 同类题迁移

每讲完一道题，都要总结一个可复用规则：
- 河流流量题：先找补给，再看汇水条件。
- 农业品质题：常看光照、昼夜温差、生长期、技术和市场。
- 工业区位题：常看原料、能源、交通、市场、劳动力、科技、政策。
- 洪涝题：分“来水多”和“排水慢”两条线。
