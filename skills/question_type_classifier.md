# 题型分类器

本文件用于快速判断一道天津地理题应该调用哪个专题、哪种答题框架。分类不是为了贴标签，而是为了决定下一步怎么推理。

## 1. 按设问动词分类

### 分析原因

识别词：
- 分析原因
- 说明成因
- 解释为什么

处理方式：
- 找材料条件。
- 把条件转成地理含义。
- 写出“条件 → 过程 → 结果”的因果链。

### 说明影响

识别词：
- 说明影响
- 分析意义
- 分析不利影响
- 评价影响

处理方式：
- 先判断影响对象：经济、社会、生态、交通、产业、居民生活。
- 正面影响和负面影响分开写。
- 影响要落到具体对象，不能只写“促进发展”。

### 提出措施

识别词：
- 提出措施
- 说明治理方法
- 指出发展方向
- 如何实现可持续发展

处理方式：
- 先找问题原因。
- 每条措施对应一个原因。
- 用“加强、完善、提高、优化、合理、因地制宜”等规范动词。

### 比较差异

识别词：
- 比较
- 与……相比
- 分别说明
- 指出差异

处理方式：
- 两个对象都要写。
- 先写差异，再写原因。
- 避免只分析其中一方。

## 2. 按专题分类

河流题：
- 关键词：流量、水位、汛期、补给、含沙量、结冰期、水能、洪涝、三角洲。
- 优先调用：`chains/river_chain.md`、`extracted/river_questions.md`。

气候题：
- 关键词：气温、降水、光照、热量、昼夜温差、风、气压带风带、季风。
- 优先调用：`skills/condition_engine.md` 中纬度、海陆位置、地形、气候条件。

农业题：
- 关键词：作物、熟制、品质、产量、灌溉、土壤、市场、冷藏保鲜。
- 常用链：自然条件 + 社会经济条件 + 品质和市场竞争力。

工业题：
- 关键词：原料、能源、交通、市场、劳动力、科技、产业基础、产业集聚。
- 常用链：区位条件 → 生产成本 / 市场联系 → 产业发展。

城市题：
- 关键词：城市化、人口、基础设施、功能区、交通节点、城市扩展。
- 常用链：地形水源交通产业 → 城市形成与发展。

生态题：
- 关键词：湿地、森林、草地、荒漠化、水土流失、生物多样性、生态修复。
- 常用链：人类活动 / 自然条件变化 → 生态功能变化 → 影响与治理。

## 3. 判断优先级

1. 先看设问动词，确定答案形式。
2. 再看关键词，确定专题。
3. 再看材料条件，选择因果链。
4. 最后生成天津风格答案。

## 4. 易错提醒

- 题目问“影响”时，不要写成“原因”。
- 题目问“措施”时，不要只分析问题。
- 题目问“比较”时，不要只写一边。
- 题目问“自然条件”时，不要写市场、交通、政策。
- 题目问“社会经济条件”时，不要写气候、地形、土壤。
