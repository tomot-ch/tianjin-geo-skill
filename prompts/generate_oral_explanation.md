# Prompt：生成老师风格的天津地理口语讲题稿

你要根据一道天津高考地理题或模拟题，生成老师个人风格的口语讲题稿。口语讲解必须自然、清楚、可听懂，但不能牺牲地理术语、材料依据和因果链，最后必须落回天津卷规范答案。

## 读取顺序

1. 先调用 `SKILL.md` 的读题流程：
   - 识别设问动词。
   - 识别限定词。
   - 提取材料条件。
   - 转化地理含义。
   - 串联因果链。
   - 生成天津卷规范答案。
2. 再参考 `skills/teaching_style.md`，使用通用讲题流程。
3. 再参考 `references/my_oral_style_summary.md`，调用个人讲题动作。
4. 再参考 `references/my_oral_phrase_bank.md`，选择服务讲题的话术。
5. 必要时读取对应 `chains/`、`templates/`、`papers/年份/` 文件。

## 输出要求

输出四部分：

1. 口语讲解。
2. 天津卷标准答案。
3. 易错提醒。
4. 可迁移规则。

## 口语讲解结构

```text
一、先翻译设问
用自然语言说明题目问什么、限定词是什么、不能答偏什么。

二、再拆材料
逐条说明题目给了哪些有效信息。

三、把材料转成地理含义
说明材料原话对应的地理术语、过程或区位条件。

四、串因果链
用“条件 → 过程 → 结果”的方式讲清楚答案为什么这样来。

五、提醒易错点
指出容易抄材料、跳步骤、写错角度、缺少限定词等问题。

六、考试里写成
给出天津卷规范答案，短句、分点、贴材料。
```

## 输出模板

```text
## 口语讲解

【按设问—材料—地理含义—因果链—答案展开】

## 天津卷标准答案

1. 【规范答案】
2. 【规范答案】
3. 【规范答案】

## 易错提醒

- 【易错点 1】
- 【易错点 2】

## 可迁移规则

- 【同类题可复用规则】
```

## 风格边界

- 可以更像微信讲题，但不能闲聊化。
- 不为了模仿风格而堆口头禅。
- 没有真实个人语料支持时，只能使用 `references/my_oral_phrase_bank.md` 中的示例占位话术，并标注为示例占位。
- 每段口语解释后，尽量给出“考试里写成……”。
- 最终答案必须符合天津卷规范表达。
