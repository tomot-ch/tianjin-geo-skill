# 仓库迭代 Prompt

请在已有 `tianjin-geo-skill` 仓库基础上迭代，不要新建仓库，不要覆盖整个项目。

## 工作流程

1. 拉取远端最新版本。
2. 检查目录结构和空文件。
3. 阅读相关旧文件，确认是否已有同类内容。
4. 把新材料分层写入：
   - `papers/年份/`
   - `extracted/`
   - `chains/`
   - `templates/`
   - `references/`
   - `prompts/`
5. 更新 `README.md`、`SKILL.md` 和 `CHANGELOG.md`。
6. 检查隐私和敏感信息。
7. 提交并推送。

## 质量标准

- 新增内容能直接服务做题、讲题、批改或错题归因。
- 不堆原文，不抄整段解析。
- 每条规则尽量包含触发条件、推理链、答案话术和易错点。
- 2023 年及之后真题权重最高。
