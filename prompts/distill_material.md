# 材料蒸馏 Prompt

请把新提供的天津高考地理材料蒸馏为可复用 skill 内容。

要求：

- 不整段复制原文。
- 保留年份、题号、材料关键词和答案依据。
- 区分原始材料、逐题蒸馏、母题归纳、因果链、答案话术。
- 若内容与旧文件重复，合并为更清晰的规则。
- 不保存学生隐私、账号密码、API key 或聊天记录原文。

输出：

1. 应新增或更新的 `papers/年份/` 文件。
2. 应补入 `extracted/` 的母题。
3. 应补入 `chains/` 的推理链。
4. 应补入 `templates/` 或 `references/` 的答案话术。
5. 本轮 CHANGELOG 条目。
