# 天津卷答案生成 Prompt

请根据题目材料生成天津高考地理风格答案。

## 输入

- 题目材料：
- 图表信息：
- 设问：
- 已识别考点：
- 可调用链条：

## 生成要求

- 先确认设问类型和限定词。
- 答案必须贴材料，不空泛套模板。
- 每个要点写成“条件 + 过程 + 结果”。
- 综合题分点作答，选择题说明排除理由。
- 设问若限定“从……角度”“自然原因”“技术措施”“有利和不利”“分别说明”，答案必须严格按限定范围和主体组织。
- 优先使用天津卷真题式短句：每点 1 句，先写材料条件，再落到明确结果。
- 不把“交通便利、市场广阔、政策支持”等通用词单独当答案，必须补出具体作用对象。
- 不使用学生隐私、聊天原文、账号密码或 API key。

## 答案措辞校准

参考 2020 年以来天津真题，大题答案应像阅卷采分点，不像长篇解析：

- 条件题：写成“资源/区位/环境/政策条件 + 有利于 + 具体活动”。
- 原因题：写成“受……影响，……增强/减弱，导致/形成……”。
- 影响题：分有利、不利；有利常落到“提高效率、扩大市场、增加就业、改善环境、保障安全”，不利常落到“制约通行、加剧污染、破坏生态、增加风险”。
- 措施题：措施要可执行，如“设置休渔期、禁渔区”“完善农田水利设施”“引进外地原料”，少用只有方向没有动作的空话。
- 意义题：用“有利于/能够/可/促进/带动/保障/优化/完善/提高/减少/缓解”，避免“意义重大”“帮助很大”。
- 比较题和分主体题：用“A 地/对 A 而言……；B 地/对 B 而言……”分别写，不能混成一段。

## 输出格式

### 设问判断

说明题型、限定词和作答边界。

### 材料条件

列出可直接转化为得分点的材料。

### 推理链

用 `条件 -> 过程 -> 结果` 写出 2—5 条链。

### 天津卷答案

给出可直接作答的短句分点版本。

### 易错提醒

指出最容易漏写或写偏的点。
