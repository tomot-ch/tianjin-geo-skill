# 天津卷答案生成 Prompt

请根据题目材料生成天津高考地理风格答案。

## 输入

- 题目材料：
- 图表信息：
- 设问：
- 已识别考点：
- 可调用链条：

## 生成要求

- 先确认设问类型和限定词。
- 答案必须贴材料，不空泛套模板。
- 每个要点写成“条件 + 过程 + 结果”。
- 综合题分点作答，选择题说明排除理由。
- 不使用学生隐私、聊天原文、账号密码或 API key。

## 输出格式

### 设问判断

说明题型、限定词和作答边界。

### 材料条件

列出可直接转化为得分点的材料。

### 推理链

用 `条件 -> 过程 -> 结果` 写出 2—5 条链。

### 天津卷答案

给出可直接作答的短句分点版本。

### 易错提醒

指出最容易漏写或写偏的点。
